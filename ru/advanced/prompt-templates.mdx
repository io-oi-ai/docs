## Обзор

Шаблоны промптов позволяют оборачивать сконвертированный Markdown инструкциями перед отправкой в AI-платформу. Вместо того чтобы каждый раз вручную набирать «Суммируй эту статью...», создайте шаблон один раз и используйте его повторно.

## Встроенные шаблоны

Web2MD поставляется с 4 готовыми к использованию шаблонами:

| Шаблон | Промпт |
|----------|--------|
| **Summarize** (Суммировать) | «Please summarize the following article concisely:» |
| **Extract Key Points** (Извлечь ключевые моменты) | «Please extract the key points and main takeaways from this article as a bullet list:» |
| **Tweet** | «Please turn the following content into a concise, engaging tweet (under 280 characters):» |
| **Q&A** (Вопросы и ответы) | «Please generate a list of questions and answers based on the following content:» |

## Как работают шаблоны

Когда вы выбираете шаблон и отправляете контент в AI, шаблон оборачивает ваш контент:

```markdown
Please summarize the following article concisely:

# Article Title

The actual converted Markdown content goes here...
```

Плейсхолдер `{content}` в вашем шаблоне заменяется на сконвертированный Markdown.

## Создание пользовательского шаблона

<Steps>
  <Step title="Откройте вкладку Convert">
    Откройте всплывающее окно Web2MD и перейдите на вкладку **Convert**.
  </Step>
  <Step title="Нажмите + Custom">
    Нажмите кнопку **«+ Custom»** рядом с выбором шаблона.
  </Step>
  <Step title="Заполните детали">
    Укажите:
    - **Name** (Название) — короткое название шаблона (например, «Code Review»)
    - **Prompt** (Промпт) — текст инструкции, используя `{content}` в качестве плейсхолдера
  </Step>
  <Step title="Сохраните и выберите">
    Сохраните шаблон. Он будет автоматически выбран в качестве активного.
  </Step>
</Steps>

### Пример пользовательского шаблона

**Название:** Code Review

**Промпт:**
```
Please review the following code and provide feedback on:
1. Code quality and best practices
2. Potential bugs or issues
3. Performance improvements

{content}
```

## Предпросмотр промпта

Наведите курсор на любой шаблон в списке выбора, чтобы увидеть предпросмотр полного промпта, который будет отправлен в AI. Это помогает убедиться, что выбран нужный шаблон, перед конвертацией.

## Выбор шаблона

Выберите активный шаблон в **списке шаблонов** на вкладке Convert. Выбранный шаблон применяется автоматически каждый раз, когда вы используете **Send to AI**.

## Редактирование и удаление шаблонов

В списке шаблонов на вкладке Convert вы можете:

- **Редактировать** название или промпт любого пользовательского шаблона
- **Удалить** пользовательские шаблоны, которые больше не нужны

<Info>
Встроенные шаблоны нельзя редактировать или удалять, но вы можете создать пользовательские шаблоны, чтобы заменить их.
</Info>
