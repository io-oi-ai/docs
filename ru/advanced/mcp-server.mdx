## Обзор

MCP-сервер Web2MD предоставляет Web2MD в виде набора инструментов, которые AI-ассистенты могут вызывать напрямую, используя [Model Context Protocol (MCP)](https://modelcontextprotocol.io) — открытый стандарт для подключения AI-моделей к внешним инструментам и источникам данных.

После настройки вы можете попросить Claude, Cursor или Windsurf преобразовать страницы, найти нужную страницу в истории преобразований или получить сохранённый Markdown, не выходя из диалога.

<Note>
  Для работы MCP-сервера требуется **API-ключ Web2MD** (тариф PRO). Создайте его в [панели управления](https://web2md.org/dashboard/api-keys).
</Note>

## Доступные инструменты

MCP-сервер предоставляет три инструмента:

| Инструмент | Описание |
|------|-------------|
| `convert_url` | Преобразование любого URL в чистый Markdown |
| `semantic_search` | Поиск по сохранённым преобразованиям на естественном языке |
| `get_conversion` | Получение полного Markdown сохранённого преобразования по ID |

### convert_url

Загружает страницу по URL и возвращает её содержимое в формате Markdown. Поддерживает те же параметры преобразования, что и REST API.

**Пример запроса:** _«Преобразуй эту ссылку в markdown: https://docs.example.com/getting-started»_

### semantic_search

Выполняет поиск по всем ранее сохранённым преобразованиям с помощью запроса на естественном языке. Возвращает список подходящих преобразований с заголовками, URL и оценкой релевантности.

**Пример запроса:** _«Найди среди моих сохранённых страниц статьи об индексации баз данных»_

### get_conversion

Возвращает полное содержимое Markdown конкретного сохранённого преобразования по его ID. Полезно для добавления целого документа в контекст после того, как он найден с помощью `semantic_search`.

**Пример запроса:** _«Покажи полное содержимое преобразования abc123»_

## Настройка

<Tabs>
  <Tab title="Claude Desktop">
    Откройте файл конфигурации Claude Desktop:

    - **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`
    - **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

    Добавьте запись `web2md` в раздел `mcpServers`:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Перезапустите Claude Desktop. Инструменты Web2MD должны появиться в списке доступных инструментов.
  </Tab>
  <Tab title="Cursor">
    Откройте файл конфигурации MCP для Cursor по пути `~/.cursor/mcp.json` и добавьте:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Перезапустите Cursor, чтобы загрузить новый сервер.
  </Tab>
  <Tab title="Windsurf">
    Откройте файл конфигурации MCP для Windsurf по пути `~/.windsurf/mcp.json` и добавьте:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Перезапустите Windsurf, чтобы загрузить новый сервер.
  </Tab>
</Tabs>

## Примеры использования

После запуска MCP-сервера вы можете использовать естественный язык в своём AI-ассистенте:

<AccordionGroup>
  <Accordion title="Преобразование страницы в Markdown">
    **Вы:** Преобразуй эту ссылку в markdown: https://react.dev/learn/thinking-in-react

    **Ассистент:** _(вызывает `convert_url`)_ Вот преобразованный Markdown для страницы «Thinking in React»...
  </Accordion>
  <Accordion title="Поиск среди сохранённых преобразований">
    **Вы:** Найди среди моих сохранённых страниц что-нибудь про аутентификацию

    **Ассистент:** _(вызывает `semantic_search`)_ Я нашёл 3 сохранённых преобразования, связанных с аутентификацией:
    1. «OAuth 2.0 Guide» — сохранено 2 дня назад
    2. «JWT Best Practices» — сохранено на прошлой неделе
    3. ...
  </Accordion>
  <Accordion title="Получение полного преобразования">
    **Вы:** Покажи полное содержимое преобразования abc123

    **Ассистент:** _(вызывает `get_conversion`)_ Вот полное содержимое в формате Markdown...
  </Accordion>
</AccordionGroup>

<Tip>
  Комбинируйте инструменты в рамках одного диалога. Например: _«Найди среди моих преобразований документацию по React, а затем покажи полное содержимое самого последнего»_.
</Tip>

## Устранение неполадок

<AccordionGroup>
  <Accordion title="Инструменты не появляются после настройки">
    Убедитесь, что вы перезапустили приложение после редактирования файла конфигурации. Проверьте, что команда `npx web2md-mcp` успешно выполняется в терминале.
  </Accordion>
  <Accordion title="Ошибки аутентификации">
    Убедитесь, что ваш `WEB2MD_API_KEY` указан верно и подписка PRO активна. Проверить ключ можно на [странице API-ключей](https://web2md.org/dashboard/api-keys).
  </Accordion>
  <Accordion title="npx не может разрешить пакет">
    Попробуйте сначала установить пакет глобально: `npm install -g web2md-mcp`, затем измените `command` на `web2md-mcp` и удалите поле `args`.
  </Accordion>
</AccordionGroup>
