## Обзор

Настройки доступны на вкладке **Settings** в попапе Web2MD, а также на странице параметров (щёлкните правой кнопкой мыши по иконке расширения → Options).

## Параметры преобразования

| Настройка | По умолчанию | Описание |
|---------|---------|-------------|
| **Include images** | Вкл | Сохранять ссылки на изображения с alt-текстом в результате Markdown |
| **Include links** | Вкл | Сохранять гиперссылки в результате. Отключите для получения обычного текста |
| **Add metadata** | Вкл | Добавлять YAML frontmatter (заголовок, исходный URL, временная метка) в результат |
| **Detect code language** | Вкл | Автоматически определять язык программирования для блоков кода |
| **Auto copy** | Выкл | Автоматически копировать Markdown в буфер обмена после преобразования |

## Параметры карточки предпросмотра

| Настройка | По умолчанию | Описание |
|---------|---------|-------------|
| **Show preview card** | Вкл | Показывать плавающую карточку предпросмотра после преобразования через сочетание клавиш/контекстное меню |
| **Card duration** | 15 с | Как долго карточки остаются видимыми: 5 с, 15 с, 30 с или Always |
| **Clear all cards** | — | Удалить все постоянные карточки (видимые в режиме Always) |

## Настройки AI

| Настройка | По умолчанию | Описание |
|---------|---------|-------------|
| **Preferred AI platform** | Claude | Платформа по умолчанию для функции Send to AI: Claude, ChatGPT или Gemini |
| **Prompt template** | Нет | Активный шаблон, применяемый при отправке в AI. [Узнать больше →](/ru/advanced/prompt-templates) |
| **Auto send to AI** | Выкл | Автоматически отправлять в AI после каждого преобразования |

## Настройки Obsidian

| Настройка | По умолчанию | Описание |
|---------|---------|-------------|
| **Enable Obsidian export** | Выкл | Показывать кнопку «Save to Obsidian». [Узнать больше →](/ru/advanced/obsidian) |
| **Vault name** | (пусто) | Имя целевого хранилища (vault) Obsidian |

## Хранение данных

Web2MD хранит ваши настройки локально с помощью `chrome.storage.local`. Настройки синхронизируются между сессиями браузера, но **не** синхронизируются между устройствами.

Никакие персональные данные или преобразованный контент не отправляются на серверы Web2MD, если только вы не используете бесплатный тариф с серверным API преобразования.
