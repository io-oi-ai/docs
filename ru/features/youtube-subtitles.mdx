## Обзор

Web2MD может извлекать субтитры из видео YouTube и конвертировать их в чистый Markdown с временными метками. Это работает как с вручную загруженными субтитрами, так и с автоматически сгенерированными (ASR).

## Что вы получаете

Конвертация видео YouTube даёт:

- **Метаданные видео** — название, канал, длительность, количество просмотров
- **Описание** — текст описания видео
- **Полную расшифровку** — субтитры, сгруппированные в абзацы по ~30 секунд с временными метками

## Поддерживаемые URL

- `youtube.com/watch?v=...`
- `youtu.be/...`
- `youtube.com/shorts/...`

## Приоритет языка

Web2MD выбирает лучшую доступную дорожку субтитров:
1. Субтитры на английском, добавленные вручную (наивысшее качество)
2. Любые субтитры, добавленные вручную
3. Автоматически сгенерированные на английском (ASR)
4. Любые автоматически сгенерированные субтитры
