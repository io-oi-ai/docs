## Обзор

Меню действий — это раздельная кнопка (split-button) во всплывающем окне расширения. Основная кнопка копирует страницу как Markdown в один клик. Нажмите на **стрелку ↑**, чтобы развернуть выпадающий список со всеми доступными действиями.

## Доступные действия

<CardGroup cols={2}>
  <Card title="Copy page" icon="copy">
    Скопировать текущую страницу как Markdown для LLM. Это действие по умолчанию.
  </Card>
  <Card title="View as Markdown" icon="arrow-up-right-from-square">
    Открыть текущую страницу как чистый текст в новой вкладке — удобно для чтения или отправки без использования AI.
  </Card>
  <Card title="Open in ChatGPT" icon="arrow-up-right-from-square">
    Конвертировать страницу и отправить её напрямую в [chatgpt.com](https://chatgpt.com), чтобы задать вопросы по её содержанию.
  </Card>
  <Card title="Open in Claude" icon="arrow-up-right-from-square">
    Конвертировать страницу и отправить её напрямую в [claude.ai](https://claude.ai), чтобы задать вопросы по её содержанию.
  </Card>
  <Card title="Open in Perplexity" icon="arrow-up-right-from-square">
    Конвертировать страницу и отправить её напрямую в [perplexity.ai](https://perplexity.ai), чтобы задать вопросы по её содержанию.
  </Card>
</CardGroup>

## Как использовать

<Steps>
  <Step title="Нажмите основную кнопку">
    Нажмите **Copy page**, чтобы мгновенно скопировать текущую страницу как Markdown. Для самого частого действия выпадающий список не требуется.
  </Step>
  <Step title="Откройте выпадающий список">
    Нажмите на **стрелку ↑** справа от кнопки, чтобы увидеть все доступные действия.
  </Step>
  <Step title="Выберите действие">
    Выберите нужное действие из меню. "View as Markdown" и варианты с AI открываются в новой вкладке автоматически.
  </Step>
</Steps>

## Зачем раздельная кнопка?

Раньше приходилось заходить в настройки, чтобы переключаться между режимами Copy, Send to AI или View. Меню действий предоставляет доступ ко всем действиям сразу — без необходимости переходить в настройки.

<Tip>
Используйте **Copy page** для быстрых сценариев с буфером обмена. Используйте AI-ярлыки, когда хотите сразу же задать вопросы по содержанию страницы.
</Tip>
