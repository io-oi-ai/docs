## Обзор

Пакетная конвертация позволяет конвертировать несколько страниц за одну операцию. Идеально подходит для исследовательских сессий, сохранённых в закладках статей или архивирования сразу нескольких страниц.

## Два режима ввода

### Вставка URL-адресов

Введите несколько URL-адресов (по одному на строку) и конвертируйте их все разом.

<Steps>
  <Step title="Откройте вкладку Batch">
    Нажмите на вкладку **Batch** во всплывающем окне Web2MD.
  </Step>
  <Step title="Вставьте ваши URL-адреса">
    Введите URL-адреса по одному на строку в текстовом поле. Можно вставить их из таблицы, заметок или любого другого источника.
  </Step>
  <Step title="Запустите конвертацию">
    Нажмите **Convert All**, чтобы начать. Каждый URL обрабатывается последовательно.
  </Step>
</Steps>

### Открытые вкладки

Выберите из текущих открытых вкладок браузера несколько страниц для конвертации.

<Steps>
  <Step title="Переключитесь в режим Open Tabs">
    Во вкладке Batch переключитесь на вид **Open Tabs**.
  </Step>
  <Step title="Выберите вкладки">
    Отметьте вкладки, которые хотите конвертировать. Перечислены все подходящие для конвертации вкладки текущего окна.
  </Step>
  <Step title="Запустите конвертацию">
    Нажмите **Convert All**, чтобы обработать все выбранные вкладки.
  </Step>
</Steps>

## Отслеживание прогресса

Во время пакетной конвертации вы увидите прогресс в реальном времени:

- **Индикатор выполнения** с процентом завершения
- Индивидуальный статус для каждого URL:

| Статус | Значение |
|--------|---------|
| Pending | Ожидает обработки |
| Converting | Конвертируется в данный момент |
| Success | Конвертация завершена |
| Error | Конвертация не удалась (например, страница недоступна) |
| Skipped | URL был некорректен или не поддерживается |

## Результаты

После завершения вы увидите сводку:

- Общее количество **успешных** конвертаций
- Общее количество **неудачных** / **пропущенных** конвертаций
- Действия: **Copy All** (все результаты вместе) или **Download All** (отдельные файлы `.md`)

Нажмите **New Batch**, чтобы начать новую пакетную конвертацию.

<Tip>
Пакетная конвертация лучше всего работает со страницами статей и записями блогов. Динамические одностраничные приложения (SPA) могут давать менее стабильные результаты.
</Tip>
