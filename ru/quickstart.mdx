## Шаг 1: Установите расширение

<Card title="Chrome Web Store" icon="chrome" href="https://chromewebstore.google.com/detail/web2md-web-to-markdown/ijmgpkkfgpijifldbjafjiapehppcbcn">
  Установите Web2MD из Chrome Web Store. Это бесплатно — регистрация для начала работы не требуется.
</Card>

После установки в панели инструментов браузера появится иконка Web2MD ( **.MD** ). Закрепите её для быстрого доступа.

## Шаг 2: Конвертируйте первую страницу

Перейдите на любую веб-страницу и выберите один из способов:

<AccordionGroup>
  <Accordion title="Нажмите на иконку расширения" icon="puzzle-piece">
    1. Нажмите на иконку Web2MD в панели инструментов
    2. Нажмите **Convert**
    3. Содержимое страницы мгновенно конвертируется в Markdown
  </Accordion>

  <Accordion title="Используйте сочетание клавиш" icon="keyboard">
    Нажмите `Alt+M` (или `Option+M` на Mac), чтобы конвертировать текущую страницу. Это работает на любой вкладке без открытия всплывающего окна.
  </Accordion>

  <Accordion title="Контекстное меню по правому клику" icon="ellipsis-vertical">
    Щёлкните правой кнопкой мыши в любом месте страницы и выберите **Copy page as Markdown**. Если у вас выделен текст, будет сконвертировано только выделение.
  </Accordion>
</AccordionGroup>

## Шаг 3: Используйте результат

После конвертации вы увидите панель результата с несколькими вариантами действий:

| Действие | Описание |
|--------|-------------|
| **Copy** | Скопировать Markdown в буфер обмена |
| **Download** | Сохранить как файл `.md` |
| **Send to AI** | Открыть Claude, ChatGPT или Gemini с этим контентом |
| **Save to Obsidian** | Отправить в ваше хранилище Obsidian (если включено) |

Также вы увидите полезную статистику: **количество слов**, **количество токенов** и **примерное время чтения**.

<Tip>
Включите **Auto copy** в настройках, чтобы Markdown автоматически копировался в буфер обмена после каждой конвертации.
</Tip>
