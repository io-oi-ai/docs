## План Free

Тариф Free включает **3 конвертации в день** — этого достаточно, чтобы попробовать Web2MD и понять, подходит ли он для вашей работы.

- Конвертации обрабатываются на сервере через API Web2MD
- Доступны все основные функции (отправка в AI, пакетная конвертация, шаблоны запросов и др.)
- Банковская карта не требуется

<Info>
При достижении бесплатного лимита вы увидите сообщение: **"Free limit reached, please upgrade to Pro."**
</Info>

## План Pro

Pro открывает **неограниченное количество конвертаций** с **обработкой на стороне клиента** — ваш контент никогда не покидает браузер.

Дополнительные возможности Pro:

- **AI-автотеги и краткое содержание** — каждая конвертация автоматически создаёт смысловые теги и краткое содержание
- **Скачивание файлов Markdown** — сохраняйте результаты конвертации как файлы `.md` на компьютер
- **Автоматические коллекции по платформам** — конвертации автоматически сортируются по исходной платформе
- **Подсчёт токенов** — точное количество токенов для планирования контекста AI
- **Умное разбиение контента** — автоматическое разбиение длинного контента на удобные для AI фрагменты
- **AI-обзоры и теги** — автоматическое создание краткого содержания и тематических тегов для сконвертированного контента
- **Коллекции и семантический поиск** — организуйте сохранённый контент в коллекции и ищите по смыслу
- **Пользовательские правила извлечения** — создавайте правила CSS-селекторов для отдельных доменов для точного извлечения контента
- **Доступ к REST API** — используйте API конвертации Web2MD программно (60 запросов/мин)
- **Интеграция MCP Server** — подключайте Web2MD к AI-агентам и инструментам автоматизации
- **Автоматизация через вебхуки** — отправляйте события конвертации во внешние сервисы
- **Инструмент CLI** — конвертируйте URL из командной строки с помощью `npx web2md`, включая пакетный режим и экспорт в хранилище Obsidian
- **Agent Bridge** — позвольте AI-агентам пакетно конвертировать URL через ваше расширение Chrome с помощью Native Messaging
- **Экспорт в виде изображения** — сохраняйте сконвертированный Markdown как стилизованные изображения без водяного знака
- **Watch Mode** — отслеживайте изменения на страницах с автоматической конвертацией или срабатыванием вебхуков
- **Compile Wiki** — создавайте готовые для Obsidian базы знаний на основе коллекций с помощью AI
- **Управление API-ключами** — создавайте и управляйте до 5 API-ключами

## Как работает конвертация в зависимости от тарифа

<AccordionGroup>
  <Accordion title="Пользователи Free" icon="user">
    1. Запускается адаптер сайта (если доступен) — это бесплатно и без ограничений
    2. Если адаптера нет, HTML страницы отправляется на серверный API Web2MD
    3. Сервер конвертирует HTML в Markdown и возвращает результат
    4. Счётчик использования увеличивается
  </Accordion>

  <Accordion title="Пользователи Pro" icon="crown">
    1. Запускается адаптер сайта (если доступен)
    2. Если адаптера нет, конвертация происходит **полностью в браузере** (на стороне клиента)
    3. Никакие данные не отправляются на сервер
    4. Никаких ограничений по использованию
  </Accordion>
</AccordionGroup>
