## Что такое Web2MD?

Web2MD — это расширение для Chrome, которое преобразует любую веб-страницу в чистый формат Markdown. Отправляете ли вы контент AI-ассистентам, сохраняете статьи в Obsidian или архивируете веб-контент — Web2MD берёт конвертацию на себя, чтобы вам не приходилось делать это вручную.

### Дополнительные возможности

- **AI-обзоры и теги** — автоматическое создание краткого содержания и тематических тегов для каждой конвертации (PRO)
- **Коллекции** — организуйте сохранённый контент в коллекции для управления знаниями (PRO)
- **Семантический поиск** — находите сохранённый контент по смыслу, а не только по ключевым словам (PRO)
- **REST API и MCP Server** — интегрируйте Web2MD в свои процессы разработки программно (PRO)
- **Инструмент CLI** — `npx web2md` конвертирует URL из командной строки. Передавайте результат в LLM, обрабатывайте пакетно или сохраняйте в хранилища Obsidian
- **Agent Bridge** — позвольте AI-агентам (Claude Code, Cursor) пакетно конвертировать Reddit и другие сайты через ваше расширение Chrome
- **Экспорт в виде изображения** — сохраняйте Markdown как стилизованное изображение в тёмной или светлой теме (PRO)
