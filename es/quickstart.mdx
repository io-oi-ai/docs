## Paso 1: Instala la extensión

<Card title="Chrome Web Store" icon="chrome" href="https://chromewebstore.google.com/detail/web2md-web-to-markdown/ijmgpkkfgpijifldbjafjiapehppcbcn">
  Instala Web2MD desde la Chrome Web Store. Es gratis, no necesitas una cuenta para empezar.
</Card>

Después de la instalación, verás el ícono de Web2MD ( **.MD** ) en la barra de herramientas de tu navegador. Fíjalo para acceder fácilmente.

## Paso 2: Convierte tu primera página

Navega a cualquier página web y elige uno de estos métodos:

<AccordionGroup>
  <Accordion title="Haz clic en el ícono de la extensión" icon="puzzle-piece">
    1. Haz clic en el ícono de Web2MD en la barra de herramientas
    2. Haz clic en **Convert**
    3. El contenido de la página se convierte a Markdown al instante
  </Accordion>

  <Accordion title="Usa el atajo de teclado" icon="keyboard">
    Presiona `Alt+M` (u `Option+M` en Mac) para convertir la página actual. Esto funciona en cualquier pestaña sin necesidad de abrir el popup.
  </Accordion>

  <Accordion title="Menú contextual (clic derecho)" icon="ellipsis-vertical">
    Haz clic derecho en cualquier parte de la página y selecciona **Copy page as Markdown**. Si tienes texto seleccionado, se convertirá solo la selección.
  </Accordion>
</AccordionGroup>

## Paso 3: Usa el resultado

Después de la conversión, verás el panel de resultado con varias opciones:

| Acción | Descripción |
|--------|-------------|
| **Copy** | Copia el Markdown al portapapeles |
| **Download** | Guarda como archivo `.md` |
| **Send to AI** | Abre Claude, ChatGPT o Gemini con el contenido |
| **Save to Obsidian** | Envía a tu bóveda de Obsidian (si está habilitado) |

También verás estadísticas útiles: **número de palabras**, **número de tokens** y **tiempo de lectura estimado**.

<Tip>
Activa **Auto copy** en la configuración para copiar automáticamente el Markdown a tu portapapeles después de cada conversión.
</Tip>
