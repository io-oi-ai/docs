## Plan Free

El plan Free incluye **3 conversiones por día**, suficiente para probar Web2MD y ver si se ajusta a tu flujo de trabajo.

- Las conversiones se procesan en el servidor a través de la API de Web2MD
- Todas las funciones principales están disponibles (Enviar a IA, conversión por lotes, plantillas de prompt, etc.)
- No se requiere tarjeta de crédito

<Info>
Cuando alcances el límite gratuito, verás un mensaje: **"Free limit reached, please upgrade to Pro."**
</Info>

## Plan Pro

Pro desbloquea **conversiones ilimitadas** con **procesamiento en el cliente**: tu contenido nunca sale de tu navegador.

Funciones adicionales de Pro:

- **Etiquetas y resumen automáticos con IA** — Cada conversión genera automáticamente etiquetas semánticas y un resumen conciso
- **Descargar archivos Markdown** — Guarda las conversiones como archivos `.md` en tu computadora
- **Colecciones automáticas por plataforma** — Las conversiones se organizan automáticamente según la plataforma de origen
- **Conteo de tokens** — Consulta el número exacto de tokens para planificar el contexto de IA
- **División inteligente de contenido** — Divide automáticamente contenido extenso en fragmentos aptos para IA
- **Resumen y etiquetas con IA** — Genera automáticamente resúmenes y etiquetas temáticas para el contenido convertido
- **Colecciones y búsqueda semántica** — Organiza el contenido guardado en colecciones y búscalo por significado
- **Reglas de extracción personalizadas** — Crea reglas de selectores CSS por dominio para una extracción de contenido precisa
- **Acceso a la API REST** — Usa la API de conversión de Web2MD de forma programática (60 solicitudes/min)
- **Integración con servidor MCP** — Conecta Web2MD con agentes de IA y herramientas de automatización
- **Automatización con webhooks** — Envía eventos de conversión a servicios externos
- **Herramienta CLI** — Convierte URLs desde la línea de comandos con `npx web2md`, incluyendo modo por lotes y exportación a bóvedas de Obsidian
- **Agent Bridge** — Permite que agentes de IA conviertan URLs por lotes a través de tu extensión de Chrome mediante Native Messaging
- **Exportar como imagen** — Guarda el Markdown convertido como imágenes con estilo, sin marca de agua
- **Modo de vigilancia** — Monitorea páginas en busca de cambios con conversión automática o disparadores de webhook
- **Compilar Wiki** — Genera bases de conocimiento listas para Obsidian impulsadas por IA a partir de colecciones
- **Gestión de claves API** — Crea y administra hasta 5 claves API

## Cómo funciona la conversión según el plan

<AccordionGroup>
  <Accordion title="Usuarios Free" icon="user">
    1. Se ejecuta el adaptador de sitio (si está disponible); esto es gratuito e ilimitado
    2. Si no hay adaptador, el HTML de la página se envía a la API del servidor de Web2MD
    3. El servidor convierte el HTML a Markdown y devuelve el resultado
    4. El contador de uso se incrementa
  </Accordion>

  <Accordion title="Usuarios Pro" icon="crown">
    1. Se ejecuta el adaptador de sitio (si está disponible)
    2. Si no hay adaptador, la conversión ocurre **completamente en el navegador** (del lado del cliente)
    3. No se envía ningún dato a ningún servidor
    4. Sin límites de uso
  </Accordion>
</AccordionGroup>
