## Étape 1 : Installer l'extension

<Card title="Chrome Web Store" icon="chrome" href="https://chromewebstore.google.com/detail/web2md-web-to-markdown/ijmgpkkfgpijifldbjafjiapehppcbcn">
  Installez Web2MD depuis le Chrome Web Store. C'est gratuit — aucun compte requis pour commencer.
</Card>

Après l'installation, vous verrez l'icône Web2MD ( **.MD** ) dans la barre d'outils de votre navigateur. Épinglez-la pour un accès rapide.

## Étape 2 : Convertir votre première page

Rendez-vous sur n'importe quelle page web, puis choisissez l'une de ces méthodes :

<AccordionGroup>
  <Accordion title="Cliquer sur l'icône de l'extension" icon="puzzle-piece">
    1. Cliquez sur l'icône Web2MD dans la barre d'outils
    2. Cliquez sur **Convert**
    3. Le contenu de la page est instantanément converti en Markdown
  </Accordion>

  <Accordion title="Utiliser le raccourci clavier" icon="keyboard">
    Appuyez sur `Alt+M` (ou `Option+M` sur Mac) pour convertir la page actuelle. Cela fonctionne sur n'importe quel onglet sans ouvrir le popup.
  </Accordion>

  <Accordion title="Menu contextuel (clic droit)" icon="ellipsis-vertical">
    Faites un clic droit n'importe où sur la page et sélectionnez **Copy page as Markdown**. Si du texte est sélectionné, seule la sélection sera convertie.
  </Accordion>
</AccordionGroup>

## Étape 3 : Utiliser le résultat

Après la conversion, vous verrez le panneau de résultat avec plusieurs options :

| Action | Description |
|--------|-------------|
| **Copy** | Copier le Markdown dans le presse-papiers |
| **Download** | Enregistrer sous forme de fichier `.md` |
| **Send to AI** | Ouvrir Claude, ChatGPT, ou Gemini avec le contenu |
| **Save to Obsidian** | Envoyer vers votre coffre Obsidian (si activé) |

Vous verrez également des statistiques utiles : **nombre de mots**, **nombre de tokens**, et **temps de lecture estimé**.

<Tip>
Activez **Auto copy** dans les paramètres pour copier automatiquement le Markdown dans votre presse-papiers après chaque conversion.
</Tip>
