## Vue d'ensemble

Lorsque vous convertissez une page, Web2MD génère automatiquement :
- **Un résumé de 1 à 2 phrases** — Une vue d'ensemble concise du contenu
- **3 à 5 tags sémantiques** — Des tags en anglais, en minuscules, pour la catégorisation

Cela se produit en arrière-plan (~1-2 secondes), propulsé par le modèle d'IA Doubao.

## Comment ça marche

<Steps>
  <Step title="Convertir une page">
    Convertissez n'importe quelle page web avec Web2MD.
  </Step>
  <Step title="L'IA analyse le contenu">
    Les ~4000 premiers caractères sont envoyés au modèle d'IA pour analyse.
  </Step>
  <Step title="Les tags apparaissent">
    Le résumé et les tags s'affichent sous le résultat de la conversion, sous forme de pastilles colorées.
  </Step>
</Steps>

## Filtrage par tag

Dans l'onglet **History**, vous pouvez filtrer vos conversions en cliquant sur un tag. Cela vous permet de retrouver rapidement tous les articles portant sur un sujet précis.

## Fonctionnalité PRO

<Info>
Les tags et le résumé générés par IA sont une fonctionnalité **PRO**. Les utilisateurs du plan gratuit voient un aperçu flouté avec une invitation à passer au plan supérieur.
</Info>
