## Aperçu

Depuis la v0.6, chaque conversion génère automatiquement un court résumé et un ensemble de tags sémantiques. Aucune configuration nécessaire — Web2MD s'occupe de tout en coulisses pour que vous puissiez parcourir votre historique d'un coup d'œil.

<Note>
Résumé IA & Tags est une fonctionnalité **réservée à Pro**. Les utilisateurs de la version gratuite voient un aperçu flouté avec une invitation à passer à Pro.
</Note>

## Comment ça marche

<Steps>
  <Step title="Convertissez une page">
    Convertissez n'importe quelle page web via le popup, le raccourci clavier ou le menu contextuel — le flux habituel.
  </Step>
  <Step title="Le résumé est généré automatiquement">
    Une fois la conversion terminée, Claude Haiku génère un résumé de 1 à 2 phrases ainsi que 3 à 5 tags sémantiques à partir du contenu extrait.
  </Step>
  <Step title="Consultez-le dans la carte d'aperçu ou l'historique">
    Le résumé et les tags apparaissent directement dans la carte d'aperçu et dans votre liste d'historique.
  </Step>
</Steps>

## Ce que vous obtenez

<CardGroup cols={2}>
  <Card title="Résumé intelligent" icon="align-left">
    Un résumé concis de 1 à 2 phrases du contenu de la page. Utile pour se rappeler rapidement de quoi parlait un article sans avoir à le relire.
  </Card>
  <Card title="Tags sémantiques" icon="tags">
    3 à 5 tags générés automatiquement qui capturent les sujets clés. Cliquez sur un tag pour filtrer votre historique par ce sujet.
  </Card>
</CardGroup>

## Où apparaissent les résumés

| Emplacement | Ce que vous voyez |
|----------|-------------|
| **Carte d'aperçu** | Résumé sous la barre de statistiques, tags sous forme de pastilles cliquables |
| **Liste d'historique** | Extrait du résumé sous chaque entrée, tags pour un filtrage rapide |
| **Recherche dans l'historique** | Cliquez sur un tag pour trouver toutes les conversions liées à ce sujet |

## Gratuit vs Pro

<Tabs>
  <Tab title="Utilisateurs Pro">
    Le résumé complet et les tags s'affichent à chaque conversion. Cliquez sur un tag pour accéder à une vue filtrée de l'historique affichant toutes les conversions correspondantes.
  </Tab>
  <Tab title="Utilisateurs gratuits">
    Le résumé et les tags sont générés mais affichés sous forme d'**aperçu flouté**. Cliquez sur la zone floutée pour voir une invitation à passer à Pro.
  </Tab>
</Tabs>

## Zéro configuration

Il n'y a rien à configurer. Vous n'avez besoin ni de fournir une clé API ni d'activer un interrupteur — Résumé IA & Tags fonctionne automatiquement à chaque conversion pour les utilisateurs Pro.

<Tip>
Les tags sont un moyen rapide de constituer une base de connaissances consultable. Au fil du temps, votre historique devient une bibliothèque indexée par sujet de tout ce que vous avez converti.
</Tip>
