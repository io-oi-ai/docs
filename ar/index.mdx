## ما هو Web2MD؟

Web2MD هو إضافة Chrome تحوّل أي صفحة ويب إلى تنسيق Markdown نظيف. سواء كنت تُغذّي مساعدي الذكاء الاصطناعي بالمحتوى، أو تحفظ المقالات في Obsidian، أو تؤرشف محتوى الويب — يتولى Web2MD عملية التحويل نيابةً عنك.

### إمكانيات إضافية

- **ملخصات ووسوم مدعومة بالذكاء الاصطناعي** — توليد تلقائي للملخصات ووسوم المواضيع لكل عملية تحويل (PRO)
- **المجموعات** — نظّم المحتوى المحفوظ في مجموعات لإدارة المعرفة (PRO)
- **البحث الدلالي** — ابحث عن المحتوى المحفوظ بالمعنى، وليس فقط بالكلمات المفتاحية (PRO)
- **REST API وMCP Server** — دمج Web2MD في مسارات عمل المطورين برمجيًا (PRO)
- **أداة CLI** — يحوّل `npx web2md` الروابط من سطر الأوامر. مرّرها إلى نماذج اللغة الكبيرة، أو عالجها دفعيًا، أو احفظها في خزائن Obsidian
- **Agent Bridge** — امنح وكلاء الذكاء الاصطناعي (Claude Code وCursor) إمكانية تحويل Reddit ومواقع أخرى دفعيًا عبر إضافة Chrome الخاصة بك
- **التصدير كصورة** — احفظ Markdown كصورة منسّقة بمظهر داكن أو فاتح (PRO)
