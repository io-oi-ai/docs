## نظرة عامة

تتيح لك قوالب التعليمات (Prompt Templates) تغليف محتوى Markdown المحوَّل بتعليمات قبل إرساله إلى منصة ذكاء اصطناعي. بدلًا من كتابة "لخّص هذا المقال..." يدويًا في كل مرة، أنشئ قالبًا مرة واحدة وأعد استخدامه.

## القوالب المدمجة

يأتي Web2MD مع 4 قوالب جاهزة للاستخدام:

| القالب | التعليمة |
|----------|--------|
| **تلخيص (Summarize)** | "يرجى تلخيص المقال التالي بإيجاز:" |
| **استخراج النقاط الرئيسية** | "يرجى استخراج النقاط الرئيسية والخلاصات الأساسية من هذا المقال كقائمة نقطية:" |
| **تغريدة (Tweet)** | "يرجى تحويل المحتوى التالي إلى تغريدة موجزة وجذابة (أقل من 280 حرفًا):" |
| **أسئلة وأجوبة** | "يرجى إنشاء قائمة من الأسئلة والأجوبة بناءً على المحتوى التالي:" |

## كيف تعمل القوالب

عند اختيار قالب وإرسال المحتوى إلى الذكاء الاصطناعي، يقوم القالب بتغليف محتواك:

```markdown
Please summarize the following article concisely:

# Article Title

The actual converted Markdown content goes here...
```

يُستبدَل العنصر النائب `{content}` في قالبك بمحتوى Markdown المحوَّل.

## إنشاء قالب مخصص

<Steps>
  <Step title="افتح تبويب Convert">
    افتح نافذة Web2MD المنبثقة وانتقل إلى تبويب **Convert**.
  </Step>
  <Step title="اضغط على + Custom">
    اضغط زر **"+ Custom"** بجانب قائمة اختيار القوالب.
  </Step>
  <Step title="أدخل التفاصيل">
    أدخل:
    - **الاسم** — اسم مختصر للقالب (مثل "مراجعة الكود")
    - **التعليمة** — نص التعليمات، مستخدمًا `{content}` كعنصر نائب
  </Step>
  <Step title="احفظ واختر">
    احفظ القالب. سيتم اختياره تلقائيًا كقالب نشط.
  </Step>
</Steps>

### مثال على قالب مخصص

**الاسم:** مراجعة الكود

**التعليمة:**
```
Please review the following code and provide feedback on:
1. Code quality and best practices
2. Potential bugs or issues
3. Performance improvements

{content}
```

## معاينة التعليمة

مرّر مؤشر الفأرة فوق أي قالب في قائمة اختيار القوالب لرؤية معاينة للتعليمة الكاملة التي سترسَل إلى الذكاء الاصطناعي. يساعدك هذا على التأكد من اختيار القالب الصحيح قبل التحويل.

## اختيار قالب

اختر قالبك النشط من **قائمة اختيار القوالب** في تبويب Convert. يُطبَّق القالب المختار تلقائيًا عند استخدام **Send to AI**.

## تعديل أو حذف القوالب

في قائمة اختيار القوالب بتبويب Convert، يمكنك:

- **تعديل** اسم أو تعليمة أي قالب مخصص
- **حذف** القوالب المخصصة التي لم تعد بحاجة إليها

<Info>
لا يمكن تعديل أو حذف القوالب المدمجة، لكن يمكنك إنشاء قوالب مخصصة لتجاوزها.
</Info>
