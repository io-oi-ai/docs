## الخطة المجانية

تتضمن الخطة المجانية **3 عمليات تحويل يوميًا** — وهو ما يكفي لتجربة Web2MD ومعرفة ما إذا كان يناسب سير عملك.

- تُعالج عمليات التحويل من جانب الخادم عبر Web2MD API
- جميع الميزات الأساسية متاحة (الإرسال إلى الذكاء الاصطناعي، التحويل الدفعي، قوالب الأوامر، إلخ)
- لا حاجة لبطاقة ائتمان

<Info>
عند وصولك إلى الحد المجاني، ستظهر لك رسالة: **"Free limit reached, please upgrade to Pro."**
</Info>

## خطة Pro

تفتح خطة Pro **عمليات تحويل غير محدودة** مع **معالجة من جانب العميل** — بحيث لا يغادر محتواك متصفحك أبدًا.

ميزات إضافية في Pro:

- **وسوم وملخص تلقائي بالذكاء الاصطناعي** — كل عملية تحويل تولّد تلقائيًا وسومًا دلالية وملخصًا موجزًا
- **تنزيل ملفات Markdown** — احفظ عمليات التحويل كملفات `.md` على جهازك
- **تصنيف تلقائي حسب المنصة** — يتم فرز عمليات التحويل تلقائيًا حسب منصة المصدر
- **عدّ الرموز (Token)** — اطّلع على العدد الدقيق للرموز للتخطيط لسياق الذكاء الاصطناعي
- **تقسيم ذكي للمحتوى** — يقسّم المحتوى الطويل تلقائيًا إلى أجزاء مناسبة للذكاء الاصطناعي
- **ملخص ووسوم بالذكاء الاصطناعي** — توليد تلقائي للملخصات ووسوم المواضيع للمحتوى المحوَّل
- **المجموعات والبحث الدلالي** — نظّم المحتوى المحفوظ في مجموعات وابحث بالمعنى
- **قواعد استخراج مخصصة** — أنشئ قواعد محددات CSS لكل نطاق لاستخراج دقيق للمحتوى
- **الوصول إلى REST API** — استخدم واجهة برمجة تحويل Web2MD برمجيًا (60 طلب/دقيقة)
- **تكامل MCP Server** — اربط Web2MD بوكلاء الذكاء الاصطناعي وأدوات الأتمتة
- **أتمتة عبر Webhook** — أرسل أحداث التحويل إلى خدمات خارجية
- **أداة CLI** — حوّل الروابط من سطر الأوامر باستخدام `npx web2md`، بما في ذلك الوضع الدفعي وتصدير خزنة Obsidian
- **Agent Bridge** — امنح وكلاء الذكاء الاصطناعي إمكانية تحويل الروابط دفعيًا عبر إضافة Chrome الخاصة بك عبر Native Messaging
- **التصدير كصورة** — احفظ Markdown المحوَّل كصور منسّقة دون علامة مائية
- **وضع المراقبة (Watch Mode)** — راقب الصفحات بحثًا عن تغييرات مع تحويل تلقائي أو محفزات Webhook
- **تجميع الويكي (Compile Wiki)** — أنشئ قواعد معرفة جاهزة لـ Obsidian مدعومة بالذكاء الاصطناعي من المجموعات
- **إدارة مفاتيح API** — أنشئ وأدر حتى 5 مفاتيح API

## كيف يعمل التحويل حسب الخطة

<AccordionGroup>
  <Accordion title="المستخدمون في الخطة المجانية" icon="user">
    1. يعمل محول الموقع (إن وُجد) — وهذا مجاني وغير محدود
    2. إذا لم يوجد محول، يُرسل HTML الصفحة إلى خادم Web2MD API
    3. يحوّل الخادم HTML إلى Markdown ويعيد النتيجة
    4. يزداد عداد الاستخدام
  </Accordion>

  <Accordion title="مستخدمو Pro" icon="crown">

    3. لا تُرسل أي بيانات إلى أي خادم
    4. لا توجد حدود استخدام
