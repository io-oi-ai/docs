## 概要

設定には、Web2MD ポップアップの **Settings** タブ、またはオプションページ(拡張機能アイコンを右クリック → Options)からアクセスできます。

## 変換オプション

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| **Include images** | オン | Markdown 出力に alt テキスト付きの画像リンクを保持します |
| **Include links** | オン | 出力にハイパーリンクを保持します。プレーンテキストにしたい場合はオフにします |
| **Add metadata** | オン | 出力に YAML フロントマター(タイトル、ソース URL、タイムスタンプ)を追加します |
| **Detect code language** | オン | コードブロックのプログラミング言語を自動検出します |
| **Auto copy** | オフ | 変換後に Markdown を自動的にクリップボードへコピーします |

## プレビューカードのオプション

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| **Show preview card** | オン | ショートカットやコンテキストメニューでの変換後に、フローティングのプレビューカードを表示します |
| **Card duration** | 15 秒 | カードの表示時間:5 秒、15 秒、30 秒、または常時表示 |
| **Clear all cards** | — | 永続カードをすべて削除します(常時表示モードで表示されるもの) |

## AI 設定

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| **Preferred AI platform** | Claude | Send to AI のデフォルトプラットフォーム:Claude、ChatGPT、または Gemini |
| **Prompt template** | なし | AI 送信時に適用されるアクティブなテンプレート。[詳しくはこちら →](/ja/advanced/prompt-templates) |
| **Auto send to AI** | オフ | 変換のたびに自動で AI に送信します |

## Obsidian 設定

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| **Enable Obsidian export** | オフ | 「Save to Obsidian」ボタンを表示します。[詳しくはこちら →](/ja/advanced/obsidian) |
| **Vault name** | (空欄) | 保存先の Obsidian Vault 名 |

## ターゲット AI モデル

<Note>v0.6 で追加</Note>

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| **Target AI model** | なし | Claude 200K、GPT-4 128K、Gemini 1M、または「なし」を選択します。設定すると、Web2MD は選択したモデルのコンテキストウィンドウに収まるようコンテンツを自動圧縮します。 |

## カスタム抽出ルール

<Note>v0.7 で追加 — PRO 限定</Note>

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| **Custom rules** | (空) | ドメインごとの CSS セレクタールールを作成し、抽出を細かく制御できます。[詳しくはこちら →](/ja/advanced/custom-rules) |

## API キー

<Note>v0.8 で追加 — PRO 限定</Note>

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| **API keys** | (なし) | REST API と MCP サーバー用の API キーを作成・管理します。アカウントあたり最大 5 個。[詳しくはこちら →](/ja/advanced/api) |

## Webhook

<Note>v0.8 で追加 — PRO 限定</Note>

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|

## ヘルプとフィードバック

| リンク | 説明 |

| **Send feedback** | サポート宛のメールを開きます |

## データの保存

Web2MD は `chrome.storage.local` を使って設定をローカルに保存します。設定はブラウザセッションをまたいで保持されますが、デバイス間では**同期されません**。

Free ユーザーがサーバーサイドの変換 API を使用する場合を除き、個人データや変換したコンテンツが Web2MD のサーバーに送信されることはありません。
