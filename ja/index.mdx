## Web2MDとは?

Web2MDは、あらゆるWebページをクリーンなMarkdown形式に変換するChrome拡張機能です。AIアシスタントへのコンテンツ入力、Obsidianへの記事保存、Webコンテンツのアーカイブなど、面倒な変換作業はすべてWeb2MDにお任せください。

### その他の機能

- **AIによる要約とタグ** — 変換のたびに要約とトピックタグを自動生成(PRO)
- **コレクション** — 保存したコンテンツをコレクションに整理してナレッジ管理(PRO)
- **セマンティック検索** — キーワードだけでなく、意味で保存コンテンツを検索(PRO)
- **REST API & MCP Server** — Web2MDを開発ワークフローにプログラムから統合(PRO)
- **CLIツール** — `npx web2md` でコマンドラインからURLを変換。LLMへのパイプ、バッチ処理、Obsidian vaultへの保存も可能
- **Agent Bridge** — AIエージェント(Claude Code、Cursor)がChrome拡張機能経由でRedditなどのサイトを一括変換
- **画像としてエクスポート** — Markdownをダーク/ライトテーマのスタイル付き画像として保存(PRO)
