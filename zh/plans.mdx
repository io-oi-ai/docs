## Free 套餐

Free 套餐每天提供 **3 次转换**——足够你试用 Web2MD，看看它是否适合你的工作流。

- 转换通过 Web2MD API 在服务端处理
- 所有核心功能都可用（发送到 AI、批量转换、提示词模板等）
- 无需绑定信用卡

<Info>
达到免费额度上限时，你会看到提示：**"Free limit reached, please upgrade to Pro."**
</Info>

## Pro 套餐

Pro 解锁**不限次转换**，并采用**客户端本地处理**——你的内容不会离开浏览器。

Pro 额外功能：

- **AI 自动标签与摘要** —— 每次转换自动生成语义标签和简明摘要
- **下载 Markdown 文件** —— 把转换结果保存为 `.md` 文件到本地
- **平台自动归类收藏集** —— 转换结果按来源平台自动分类
- **Token 统计** —— 精确的 token 数，方便规划 AI 上下文
- **智能内容分块** —— 自动把长内容拆分成适合 AI 的分块
- **AI 摘要与标签** —— 为转换后的内容自动生成摘要和主题标签
- **收藏集与语义搜索** —— 把保存的内容整理成收藏集，并按含义搜索
- **自定义提取规则** —— 按域名创建 CSS 选择器规则，实现精准内容提取
- **REST API 访问** —— 以编程方式调用 Web2MD 的转换 API（60 req/min）
- **MCP Server 集成** —— 把 Web2MD 接入 AI Agent 和自动化工具
- **Webhook 自动化** —— 把转换事件推送到外部服务
- **CLI 工具** —— 用 `npx web2md` 在命令行转换 URL，支持批量模式和导出到 Obsidian 库
- **Agent Bridge** —— 让 AI Agent 通过 Native Messaging 经由你的 Chrome 扩展批量转换 URL
- **导出为图片** —— 把转换后的 Markdown 保存为精美图片，且不带水印
- **Watch Mode** —— 监控页面变化，自动触发转换或 Webhook
- **Compile Wiki** —— 从收藏集生成 AI 驱动、可直接导入 Obsidian 的知识库
- **API Key 管理** —— 创建并管理最多 5 个 API key

## 各套餐的转换流程

<AccordionGroup>
  <Accordion title="Free 用户" icon="user">
    1. 先运行站点适配器（如果有）——这一步免费且不限次
    2. 没有适配器时，页面 HTML 会发送到 Web2MD 的服务端 API
    3. 服务端把 HTML 转换为 Markdown 并返回结果
    4. 用量计数 +1
  </Accordion>

  <Accordion title="Pro 用户" icon="crown">
    1. 先运行站点适配器（如果有）
    2. 没有适配器时，转换**完全在浏览器内**完成（客户端）
    3. 不向任何服务器发送数据
    4. 没有用量限制
  </Accordion>
</AccordionGroup>
