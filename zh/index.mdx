## Web2MD 是什么？

Web2MD 是一款 Chrome 扩展，可以把任意网页转换成干净的 Markdown 格式。无论你是要把内容喂给 AI 助手、把文章存进 Obsidian，还是做网页内容归档——转换这件事交给 Web2MD 就好。

### 更多能力

- **AI 摘要与标签** —— 每次转换自动生成摘要和主题标签（PRO）
- **收藏集** —— 把保存的内容整理成收藏集，做知识管理（PRO）
- **语义搜索** —— 按含义查找已保存的内容，不只是关键词匹配（PRO）
- **REST API 与 MCP Server** —— 以编程方式把 Web2MD 接入你的开发工作流（PRO）
- **CLI 工具** —— `npx web2md` 在命令行转换 URL。可以管道输出给 LLM、批量处理，或保存到 Obsidian 库
- **Agent Bridge** —— 让 AI Agent（Claude Code、Cursor）通过你的 Chrome 扩展批量转换 Reddit 等站点
- **导出为图片** —— 把 Markdown 保存为带深色/浅色主题的精美图片（PRO）
