## 第一步：安装扩展

<Card title="Chrome Web Store" icon="chrome" href="https://chromewebstore.google.com/detail/web2md-web-to-markdown/ijmgpkkfgpijifldbjafjiapehppcbcn">
  从 Chrome Web Store 安装 Web2MD。免费使用，上手无需注册账号。
</Card>

安装完成后，浏览器工具栏会出现 Web2MD 图标（**.MD**）。建议把它固定到工具栏，方便随时使用。

## 第二步：转换你的第一个页面

打开任意网页，然后选择以下任一方式：

<AccordionGroup>
  <Accordion title="点击扩展图标" icon="puzzle-piece">
    1. 点击工具栏中的 Web2MD 图标
    2. 点击 **Convert**
    3. 页面内容会立刻转换为 Markdown
  </Accordion>

  <Accordion title="使用键盘快捷键" icon="keyboard">
    按 `Alt+M`（Mac 上是 `Option+M`）转换当前页面。任意标签页都能用，无需打开弹窗。
  </Accordion>

  <Accordion title="右键菜单" icon="ellipsis-vertical">
    在页面任意位置右键，选择 **Copy page as Markdown**。如果你选中了文本，则只转换选中部分。
  </Accordion>
</AccordionGroup>

## 第三步：使用转换结果

转换完成后，结果面板会提供以下选项：

| 操作 | 说明 |
|--------|-------------|
| **Copy** | 把 Markdown 复制到剪贴板 |
| **Download** | 保存为 `.md` 文件 |
| **Send to AI** | 打开 Claude、ChatGPT 或 Gemini 并带上内容 |
| **Save to Obsidian** | 发送到你的 Obsidian 库（需先开启） |

你还能看到实用的统计信息：**字数**、**token 数** 和 **预计阅读时长**。

<Tip>
在设置中开启 **Auto copy**，每次转换后 Markdown 会自动复制到剪贴板。
</Tip>
