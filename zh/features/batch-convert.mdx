---

## 概述

批量转换让你在一次操作中转换多个页面。非常适合做资料调研、处理收藏的文章，或一次性归档多个页面。

## 两种输入方式

### 粘贴 URL

输入多个 URL（每行一个），一次性全部转换。

<Steps>
  <Step title="打开 Batch 标签页">
    在 Web2MD 弹窗中点击 **Batch** 标签页。
  </Step>
  <Step title="粘贴 URL">
    在文本框中输入 URL，每行一个。可以从表格、笔记或任何来源直接粘贴。
  </Step>
  <Step title="开始转换">
    点击 **Convert All** 开始。每个 URL 会依次处理。
  </Step>
</Steps>

### 已打开的标签页

从当前打开的浏览器标签页中选择，一次转换多个页面。

<Steps>
  <Step title="切换到 Open Tabs 模式">
    在 Batch 标签页中，切换到 **Open Tabs** 视图。
  </Step>
  <Step title="选择标签页">
    勾选想要转换的标签页。当前窗口中所有可转换的标签页都会列出。
  </Step>
  <Step title="开始转换">
    点击 **Convert All**，处理所有选中的标签页。
  </Step>
</Steps>

## 进度跟踪

批量转换过程中，你可以看到实时进度：

- 带百分比的**进度条**
- 每个 URL 的独立状态：

| 状态 | 含义 |
|--------|---------|
| Pending | 等待处理 |
| Converting | 正在转换 |
| Success | 转换完成 |
| Error | 转换失败（例如页面无法访问） |
| Skipped | URL 无效或不受支持 |

## 结果

完成后会显示一份汇总：

- **成功**转换的总数
- **失败** / **跳过**的总数
- 可执行操作：**Copy All**（合并复制全部结果）或 **Download All**（下载单独的 `.md` 文件）

点击 **New Batch** 可以开始新一轮批量转换。

<Tip>
批量转换最适合文章页和博客文章。动态单页应用的转换结果可能不够稳定。
</Tip>
