## 概览

选区工具栏（v0.6）会在你于任意网页上选中文本时弹出一个浮动操作栏。选中一段文字、一段代码，或整个章节——然后复制为 Markdown、追加到缓冲区，或直接发送给 AI。

## 工作流程

<Steps>
  <Step title="在页面上选中文本">
    用鼠标或键盘在网页上选中任意文本，浮动工具栏会出现在选区附近。
  </Step>
  <Step title="选择操作">
    从工具栏的三个操作中选一个：**Copy as Markdown**、**Append to buffer** 或 **Send to AI**。
  </Step>
  <Step title="得到干净的输出">
    Web2MD 把你的选区转换成结构良好的 Markdown，保留格式、链接和代码块。
  </Step>
</Steps>

## 工具栏操作

<CardGroup cols={3}>
  <Card title="复制为 Markdown" icon="copy">
    即刻把选中的文本转换为 Markdown 并复制到剪贴板。
  </Card>
  <Card title="追加到缓冲区" icon="layer-plus">
    把选区加入一个持续累积的缓冲区。在页面各处选取多个片段，合并成一份 Markdown 文档。
  </Card>
  <Card title="发送到 AI" icon="paper-plane">
    把选中的内容直接发送到你偏好的 AI 平台（Claude、ChatGPT 或 Gemini）。
  </Card>
</CardGroup>

## 追加模式

当你需要从页面的不同位置——甚至不同页面——摘取内容时，追加模式非常好用。

1. 选中一段内容，点击 **Append to buffer**
2. 滚动到另一段，再次追加
3. 按需重复
4. 打开 Web2MD 弹窗，查看合并后的 Markdown 文档

最终输出中，每个追加的片段之间会用水平分隔线（`---`）隔开。

<Tip>
追加模式特别适合从长篇文档页或论文里只挑重点章节。
</Tip>

## 智能选区扩展

Web2MD 会自动把你的选区扩展到完整的结构单元：

| 你选中的是… | Web2MD 会扩展为… |
|---------------|---------------------|
| 段落的一部分 | 整个段落 |
| 代码块的一部分 | 完整代码块（含语言标记） |
| 列表项的一部分 | 完整列表项 |
| 跨元素的文本 | 所有被完整或部分选中的块 |

这保证了 Markdown 输出始终结构完整——不会出现断掉的段落或渲染一半的代码块。

## 兼容性

选区工具栏适用于所有网站。它以轻量浮动元素的形式注入，不会干扰页面自身的 UI。

<Info>
工具栏只在选中文本时出现。点击其他地方或清除选区后，它会自动消失。
</Info>
