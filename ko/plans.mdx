## Free 요금제

Free 요금제는 **하루 3회 변환**을 제공합니다 — Web2MD를 체험하고 워크플로우에 맞는지 확인하기에 충분합니다.

- 변환은 Web2MD API를 통해 서버 사이드에서 처리됩니다
- 핵심 기능(AI로 전송, 일괄 변환, 프롬프트 템플릿 등)을 모두 사용할 수 있습니다
- 신용카드가 필요하지 않습니다

<Info>
무료 한도에 도달하면 다음 메시지가 표시됩니다: **"Free limit reached, please upgrade to Pro."**
</Info>

## Pro 요금제

Pro는 **클라이언트 사이드 처리**로 **무제한 변환**을 제공합니다 — 콘텐츠가 브라우저를 벗어나지 않습니다.

추가 Pro 기능:

- **AI 자동 태그 및 요약** — 변환할 때마다 의미 기반 태그와 간결한 요약을 자동 생성합니다
- **Markdown 파일 다운로드** — 변환 결과를 `.md` 파일로 컴퓨터에 저장합니다
- **자동 플랫폼 컬렉션** — 변환 결과가 출처 플랫폼별로 자동 분류됩니다
- **토큰 카운팅** — AI 컨텍스트 계획을 위한 정확한 토큰 수를 확인합니다
- **스마트 콘텐츠 분할** — 긴 콘텐츠를 AI 친화적인 청크로 자동 분할합니다
- **AI 요약 및 태그** — 변환된 콘텐츠의 요약과 주제 태그를 자동 생성합니다
- **컬렉션 및 시맨틱 검색** — 저장한 콘텐츠를 컬렉션으로 정리하고 의미로 검색합니다
- **커스텀 추출 규칙** — 도메인별 CSS 셀렉터 규칙을 만들어 정확한 콘텐츠 추출이 가능합니다
- **REST API 액세스** — Web2MD의 변환 API를 프로그래밍 방식으로 사용합니다 (분당 60회)
- **MCP Server 연동** — Web2MD를 AI 에이전트 및 자동화 도구에 연결합니다
- **웹훅 자동화** — 변환 이벤트를 외부 서비스로 전달합니다
- **CLI 도구** — `npx web2md`로 커맨드라인에서 URL을 변환하며, 배치 모드와 Obsidian vault 내보내기도 지원합니다
- **Agent Bridge** — AI 에이전트가 Native Messaging을 통해 Chrome 확장 프로그램으로 URL을 일괄 변환하도록 합니다
- **이미지로 내보내기** — 변환된 Markdown을 워터마크 없이 스타일 이미지로 저장합니다
- **Watch Mode** — 자동화된 변환 또는 웹훅 트리거로 페이지 변경 사항을 모니터링합니다
- **Compile Wiki** — 컬렉션에서 AI 기반의 Obsidian용 지식 베이스를 생성합니다
- **API 키 관리** — 최대 5개의 API 키를 생성하고 관리합니다

## 요금제별 변환 동작 방식

<AccordionGroup>
  <Accordion title="Free 사용자" icon="user">
    1. 사이트 어댑터가 실행됩니다 (지원하는 경우) — 무료이며 무제한입니다
    2. 어댑터가 없으면 페이지 HTML이 Web2MD의 서버 API로 전송됩니다
    3. 서버가 HTML을 Markdown으로 변환해 결과를 반환합니다
    4. 사용량 카운터가 증가합니다
  </Accordion>

  <Accordion title="Pro 사용자" icon="crown">
    1. 사이트 어댑터가 실행됩니다 (지원하는 경우)
    2. 어댑터가 없으면 변환이 **브라우저 안에서 전부** 이루어집니다 (클라이언트 사이드)
    3. 어떤 데이터도 서버로 전송되지 않습니다
    4. 사용량 제한이 없습니다
  </Accordion>
</AccordionGroup>
