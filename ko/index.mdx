## Web2MD란?

Web2MD는 어떤 웹페이지든 깔끔한 Markdown 형식으로 변환해주는 Chrome 확장 프로그램입니다. AI 어시스턴트에 콘텐츠를 전달하든, Obsidian에 글을 저장하든, 웹 콘텐츠를 아카이빙하든 — Web2MD가 변환을 대신 처리해줍니다.

### 더 많은 기능

- **AI 기반 요약 및 태그** — 변환할 때마다 요약과 주제 태그를 자동 생성합니다 (PRO)
- **컬렉션** — 저장한 콘텐츠를 컬렉션으로 정리해 지식 관리에 활용하세요 (PRO)
- **시맨틱 검색** — 키워드가 아닌 의미로 저장된 콘텐츠를 찾을 수 있습니다 (PRO)
- **REST API & MCP Server** — Web2MD를 개발 워크플로우에 프로그래밍 방식으로 통합하세요 (PRO)
- **CLI 도구** — `npx web2md`로 커맨드라인에서 URL을 변환합니다. LLM으로 파이프하거나, 일괄 처리하거나, Obsidian vault에 저장할 수 있습니다
- **Agent Bridge** — Claude Code, Cursor 같은 AI 에이전트가 Chrome 확장 프로그램을 통해 Reddit 등의 사이트를 일괄 변환하도록 하세요
- **이미지로 내보내기** — 다크/라이트 테마가 적용된 스타일 이미지로 Markdown을 저장하세요 (PRO)
