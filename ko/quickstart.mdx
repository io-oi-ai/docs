## 1단계: 확장 프로그램 설치

<Card title="Chrome Web Store" icon="chrome" href="https://chromewebstore.google.com/detail/web2md-web-to-markdown/ijmgpkkfgpijifldbjafjiapehppcbcn">
  Chrome Web Store에서 Web2MD를 설치하세요. 무료이며, 시작하는 데 계정이 필요하지 않습니다.
</Card>

설치 후 브라우저 툴바에 Web2MD 아이콘( **.MD** )이 표시됩니다. 쉽게 접근할 수 있도록 고정해두세요.

## 2단계: 첫 페이지 변환하기

원하는 웹페이지로 이동한 다음, 아래 방법 중 하나를 선택하세요.

<AccordionGroup>
  <Accordion title="확장 프로그램 아이콘 클릭" icon="puzzle-piece">
    1. 툴바에서 Web2MD 아이콘을 클릭합니다
    2. **Convert**를 클릭합니다
    3. 페이지 콘텐츠가 즉시 Markdown으로 변환됩니다
  </Accordion>

  <Accordion title="키보드 단축키 사용" icon="keyboard">
    현재 페이지를 변환하려면 `Alt+M`(Mac에서는 `Option+M`)을 누르세요. 팝업을 열지 않고도 어떤 탭에서든 동작합니다.
  </Accordion>

  <Accordion title="우클릭 컨텍스트 메뉴" icon="ellipsis-vertical">
    페이지 어디서든 우클릭한 뒤 **Copy page as Markdown**을 선택하세요. 텍스트를 선택한 상태라면 선택 영역만 변환됩니다.
  </Accordion>
</AccordionGroup>

## 3단계: 결과 활용하기

변환이 끝나면 여러 옵션이 담긴 결과 패널이 표시됩니다.

| 동작 | 설명 |
|--------|-------------|
| **Copy** | Markdown을 클립보드에 복사합니다 |
| **Download** | `.md` 파일로 저장합니다 |
| **Send to AI** | Claude, ChatGPT, Gemini를 콘텐츠와 함께 엽니다 |
| **Save to Obsidian** | Obsidian vault로 전송합니다 (활성화된 경우) |

**단어 수**, **토큰 수**, **예상 읽기 시간** 같은 유용한 통계도 확인할 수 있습니다.

<Tip>
설정에서 **Auto copy**를 활성화하면 변환할 때마다 Markdown이 클립보드에 자동으로 복사됩니다.
</Tip>
