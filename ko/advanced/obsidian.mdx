## 개요

Web2MD는 `obsidian://` URI 프로토콜을 사용해 변환된 Markdown을 [Obsidian](https://obsidian.md)으로 바로 보낼 수 있습니다. 브라우저를 벗어나지 않고도 웹 콘텐츠를 보관함에 저장할 수 있습니다.

## 설정

<Steps>
  <Step title="Obsidian 연동 활성화하기">
    **Settings → Obsidian**으로 이동해 **Enable Obsidian export**를 켭니다.
  </Step>
  <Step title="보관함 이름 설정하기 (선택 사항)">
    Obsidian 보관함이 여러 개인 경우, 특정 보관함을 지정하려면 보관함 이름을 입력하세요. 비워두면 기본 보관함을 사용합니다.
  </Step>
  <Step title="Obsidian이 실행 중인지 확인하기">
    URI 프로토콜이 동작하려면 컴퓨터에서 Obsidian이 열려 있어야 합니다.
  </Step>
</Steps>

## 사용 방법

페이지를 변환한 후:

1. 결과 패널에서 **Save to Obsidian** 버튼을 클릭합니다
2. Obsidian이 열리고 Markdown 콘텐츠로 새 노트가 생성됩니다
3. 노트 제목은 페이지 제목으로 설정됩니다

## 작동 방식

Web2MD는 Obsidian의 [URI 프로토콜](https://help.obsidian.md/Extending+Obsidian/Obsidian+URI)을 사용합니다.

```
obsidian://new?vault=MyVault&name=Article%20Title&content=...
```

이를 통해 지정된 보관함에 변환된 Markdown을 내용으로 하는 새 노트가 생성됩니다.

<Warning>
`obsidian://` 프로토콜은 URL 길이 제한이 있습니다. 매우 긴 글은 잘릴 수 있습니다. 긴 콘텐츠의 경우 **Copy**를 사용해 Obsidian에 직접 붙여넣는 것을 권장합니다.
</Warning>

## 설정 항목

| 설정 | 설명 |
|---------|-------------|
| **Enable Obsidian export** | "Save to Obsidian" 버튼 표시/숨김 |
| **Vault name** | 대상 보관함 (비워두면 기본 보관함 사용) |
</content>
