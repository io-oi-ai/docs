## 개요

Web2MD CLI를 사용하면 터미널에서 바로 어떤 URL이든 깔끔한 Markdown으로 변환할 수 있습니다. 브라우저를 열지 않고도 출력을 LLM에 파이프하거나, URL 목록을 일괄 처리하거나, Obsidian vault에 콘텐츠를 가져올 수 있습니다.

```bash
npx web2md https://example.com/article
```

<Note>
  CLI를 사용하려면 **Node.js 18+**가 필요합니다. `node -v`로 버전을 확인하세요.
</Note>

## 설치

별도 설치는 필요 없습니다 — `npx`로 바로 실행하세요.

```bash
npx web2md <url> [options]
```

또는 더 빠른 시작을 위해 전역으로 설치할 수 있습니다.

```bash
npm install -g web2md
web2md <url> [options]
```

Web2MD CLI는 설정에 따라 세 가지 모드로 동작합니다.

## 플래그

| 플래그 | 설명 |
|------|-------------|
| `--no-images` | 출력에서 이미지 참조 제거 |
| `--no-links` | 출력에서 하이퍼링크 제거 |
| `--meta` | YAML frontmatter 추가 (title, source, wordCount, tokenCount, readingTime, date) |
| `--json` | JSON `{ markdown, metadata }` 형식으로 출력 |
| `-o, --output <file>` | 출력을 파일에 저장 |
| `--output-dir <dir>` | 각 URL을 별도의 `.md` 파일로 저장 |
| `--batch <file>` | 파일에서 URL 목록 읽기 (한 줄에 하나씩, `#`은 주석) |
| `--vault <dir>` | Obsidian vault 모드: `<dir>/raw/`에 저장하고 `<dir>/INDEX.md`를 업데이트 |
| `--concurrency <n>` | 최대 병렬 요청 수 (기본값: 3, 최대: 20) |
| `--bridge` | JS 렌더링 또는 로그인 보호 사이트에 Chrome 확장 프로그램 사용 |
| `-q, --quiet` | 진행 메시지 숨기기 |

## 환경 변수

| 변수 | 설명 |
|----------|-------------|
| `WEB2MD_API_KEY` | Reddit 및 제한된 사이트용 API 키 (`w2m_xxx`) |
| `WEB2MD_API_URL` | API 기본 URL 재정의 |
| `WEB2MD_EXTENSION_ID` | `--bridge` 모드용 Chrome 확장 프로그램 ID 재정의 |

<Tip>
  세션이 바뀌어도 유지되도록 셸 프로필(`~/.zshrc` 또는 `~/.bashrc`)에 다음을 추가하세요.
  ```bash
  export WEB2MD_API_KEY="w2m_your_key_here"
  ```
</Tip>

## 사용 예시

### 기본 변환

```bash
npx web2md https://example.com/article
```

Markdown을 표준 출력(stdout)으로 출력합니다.

### LLM에 파이프하기

```bash
npx web2md https://react.dev/learn/thinking-in-react | llm "Summarize this page"
```

```bash
npx web2md https://docs.python.org/3/tutorial/classes.html | claude "Explain the key concepts"
```

### 파일로 저장

```bash
npx web2md https://example.com/article -o article.md
```

```bash
npx web2md https://example.com/article --meta -o article.md
```

`--meta` 플래그는 title, source URL, word count, token count, reading time, date가 포함된 YAML frontmatter를 앞에 추가합니다.

### 파일에서 일괄 처리

`urls.txt` 파일을 만듭니다.

```text
# Research papers
https://arxiv.org/abs/2301.00001
https://arxiv.org/abs/2301.00002

# Blog posts
https://example.com/blog/post-1
https://example.com/blog/post-2
```

그런 다음 실행합니다.

```bash
npx web2md --batch urls.txt --output-dir ./research --concurrency 5
```

각 URL은 `./research` 디렉터리에 별도의 `.md` 파일로 저장됩니다.

### Obsidian vault로 가져오기

```bash
npx web2md --batch urls.txt --vault ~/Documents/MyVault
```

이 명령은 각 페이지를 `~/Documents/MyVault/raw/`에 저장하고, 변환된 모든 페이지의 링크로 `~/Documents/MyVault/INDEX.md`를 업데이트합니다.

### API 키로 Reddit 변환하기

```bash
export WEB2MD_API_KEY="w2m_your_key_here"
npx web2md https://www.reddit.com/r/LocalLLaMA/comments/example
```

<Warning>
  Reddit은 유효한 API 키가 필요합니다. 키가 없으면 Reddit의 봇 제한 정책으로 인해 Reddit URL 변환이 실패합니다.
</Warning>

### Bridge 모드

Chrome 확장 프로그램을 이용해 JS 렌더링 페이지나 로그인 보호 페이지를 처리합니다.

```bash
npx web2md --bridge https://app.example.com/dashboard
```

<Note>
  Bridge 모드를 사용하려면 Web2MD Chrome 확장 프로그램이 설치되어 있고 Chrome이 실행 중이어야 합니다. CLI는 Chrome의 네이티브 메시징 프로토콜을 통해 확장 프로그램과 통신합니다.
</Note>

### JSON 출력

```bash
npx web2md --json https://example.com/article
```

구조화된 출력을 반환합니다.

```json
{
  "markdown": "# Article Title\n\nContent here...",
  "metadata": {
    "title": "Article Title",
    "source": "https://example.com/article",
    "wordCount": 1250,
    "tokenCount": 1680,
    "readingTime": 5,
    "date": "2026-04-11T10:30:00.000Z"
  }
}
```

프로그래밍 방식으로 활용하거나 `jq`에 파이프할 때 유용합니다.

```bash
npx web2md --json https://example.com/article | jq '.metadata.tokenCount'
```

## 최적화된 사이트

Web2MD에는 다음 사이트를 위한 내장 어댑터가 포함되어 있어 일반 변환보다 더 깔끔한 출력을 만들어냅니다.

<AccordionGroup>
  <Accordion title="최적화 지원 사이트">
    - **Wikipedia** — 깔끔한 기사 추출, 인포박스 처리
    - **arXiv** — 논문 초록 및 메타데이터
    - **Hacker News** — 댓글이 포함된 스레드
    - **GitHub** — Issues 및 Pull Requests
    - **Stack Overflow** — 질문과 답변
    - **dev.to** — 블로그 게시물
    - **Medium** — 기사 (페이월 미리보기 우회)
    - **Substack** — 뉴스레터 게시물
    - **OpenAI Docs** — 문서 페이지
    - **Mintlify 기반 문서** — Mintlify로 구축된 문서 사이트
    - **Reddit** — 게시물 및 댓글 (API 키 필요)
  </Accordion>
</AccordionGroup>

  </Accordion>

    ```bash
    npx web2md --no-images --no-links https://example.com/article | llm "Analyze this"
    ```
    LLM에 파이프할 때 토큰 사용량을 줄이기 위해 이미지와 링크를 제거합니다.
