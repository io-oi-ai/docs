## 개요

일괄 변환(Batch Convert)을 사용하면 한 번의 작업으로 여러 페이지를 변환할 수 있습니다. 리서치 세션, 북마크한 기사, 여러 페이지를 한꺼번에 아카이빙할 때 특히 유용합니다.

## 두 가지 입력 방식

### URL 붙여넣기

여러 URL을 (한 줄에 하나씩) 입력하면 한 번에 모두 변환할 수 있습니다.

<Steps>
  <Step title="Batch 탭을 여세요">
    Web2MD 팝업에서 **Batch** 탭을 클릭합니다.
  </Step>
  <Step title="URL을 붙여넣으세요">
    텍스트 영역에 URL을 한 줄에 하나씩 입력합니다. 스프레드시트, 메모 등 어디서든 붙여넣을 수 있습니다.
  </Step>
  <Step title="변환을 시작하세요">
    **Convert All**을 클릭하면 시작됩니다. 각 URL은 순차적으로 처리됩니다.
  </Step>
</Steps>

### 열린 탭 사용

현재 열려 있는 브라우저 탭에서 선택해 여러 페이지를 한 번에 변환할 수 있습니다.

<Steps>
  <Step title="Open Tabs 모드로 전환하세요">
    Batch 탭에서 **Open Tabs** 보기로 전환합니다.
  </Step>
  <Step title="탭을 선택하세요">
    변환하려는 탭에 체크합니다. 현재 창에서 변환 가능한 모든 탭이 목록에 표시됩니다.
  </Step>
  <Step title="변환을 시작하세요">
    **Convert All**을 클릭하면 선택한 모든 탭이 처리됩니다.
  </Step>
</Steps>

## 진행 상황 추적

일괄 변환 중에는 실시간으로 진행 상황을 확인할 수 있습니다.

- 완료율을 보여주는 **진행률 바**
- 각 URL별 개별 상태:

| 상태 | 의미 |
|--------|---------|
| Pending | 처리 대기 중 |
| Converting | 변환 진행 중 |
| Success | 변환 완료 |
| Error | 변환 실패(예: 페이지 접근 불가) |
| Skipped | URL이 유효하지 않거나 지원되지 않음 |

## 결과

변환이 완료되면 다음과 같은 요약을 확인할 수 있습니다.

- **성공한** 변환 총 개수
- **실패** / **건너뛴** 변환 총 개수
- 액션: **Copy All**(모든 결과를 합쳐서) 또는 **Download All**(개별 `.md` 파일로)

**New Batch**를 클릭하면 새로운 일괄 변환을 시작할 수 있습니다.

<Tip>
일괄 변환은 기사 페이지나 블로그 글에서 가장 좋은 결과를 냅니다. 동적인 단일 페이지 앱(SPA)에서는 결과가 다소 일관되지 않을 수 있습니다.
</Tip>
