description: "변환 후 빠른 복사와 전송 작업을 위한 페이지 내 플로팅 카드입니다."
---

## 개요

키보드 단축키(`Alt+M`) 또는 컨텍스트 메뉴를 사용해 페이지를 변환하면, 페이지 우측 상단에 플로팅 미리보기 카드가 나타납니다. 팝업을 열지 않고도 바로 작업을 진행할 수 있습니다.

## 카드 기능

각 미리보기 카드는 다음 정보를 보여줍니다:

- 변환된 콘텐츠의 **페이지 제목**
- **단어 수** 요약
- 카드를 선택/해제하는 **체크박스**
- 닫기를 위한 **닫기 버튼**

### 하단 액션 바

하나 이상의 카드가 선택되면:

| 작업 | 설명 |
|--------|-------------|
| **Copy All** | 선택된 모든 카드의 Markdown을 클립보드에 복사 |
| **Send All** | 선택된 모든 콘텐츠를 원하는 AI 플랫폼으로 전송 |

## 여러 개의 카드

한 번에 최대 **5개의 미리보기 카드**를 쌓을 수 있습니다. 여러 페이지를 순서대로 변환할 때 유용하며, 변환할 때마다 새 카드가 추가됩니다.

## 표시 지속 시간

**Settings → Show preview card**에서 카드가 얼마나 오래 표시될지 설정할 수 있습니다:

| 지속 시간 | 동작 |
|----------|------|
| **5초** | 5초 후 카드가 자동으로 닫힘 |
| **15초** | 15초 후 카드가 자동으로 닫힘 |
| **30초** | 30초 후 카드가 자동으로 닫힘 |
| **항상** | 수동으로 닫을 때까지 카드가 유지됩니다. 페이지 새로고침 후에도 유지됩니다. |

<Info>
**항상** 모드에서는 카드가 `chrome.storage.local`에 저장되어 페이지를 새로고침해도 다시 나타납니다. 카드를 제거하려면 Settings의 **Clear all cards**를 사용하세요.
</Info>

## 미리보기 카드 켜기/끄기

**Settings → Show preview card**에서 미리보기 카드를 완전히 비활성화할 수 있습니다. 비활성화하면 단축키/컨텍스트 메뉴를 통한 변환은 계속 작동하지만 시각적 표시는 나타나지 않습니다.
