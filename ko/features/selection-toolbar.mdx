## 개요

선택 툴바(Selection Toolbar, v0.6)는 웹페이지에서 텍스트를 드래그해 선택할 때마다 떠 있는 액션 바를 제공합니다. 문단, 코드 스니펫, 전체 섹션 등 무엇이든 선택한 뒤 Markdown으로 복사하거나, 버퍼에 추가하거나, AI로 바로 전송할 수 있습니다.

## 작동 방식

<Steps>
  <Step title="어떤 페이지에서든 텍스트를 선택하세요">
    마우스나 키보드로 웹페이지의 텍스트를 드래그해 선택합니다. 선택 영역 근처에 떠 있는 툴바가 나타납니다.
  </Step>
  <Step title="액션을 선택하세요">
    툴바에서 **Copy as Markdown**, **Append to buffer**, **Send to AI** 중 하나를 선택합니다.
  </Step>
  <Step title="깔끔한 결과물을 받으세요">
    Web2MD가 서식, 링크, 코드 블록을 유지한 채로 선택 영역을 구조화된 Markdown으로 변환합니다.
  </Step>
</Steps>

## 툴바 액션

<CardGroup cols={3}>
  <Card title="Copy as Markdown" icon="copy">
    선택한 텍스트를 즉시 Markdown으로 변환해 클립보드에 복사합니다.
  </Card>
  <Card title="Append to Buffer" icon="layer-plus">
    선택 영역을 진행 중인 버퍼에 추가합니다. 페이지 곳곳의 여러 조각을 선택해 하나의 Markdown 문서로 합칠 수 있습니다.
  </Card>
  <Card title="Send to AI" icon="paper-plane">
    선택한 콘텐츠를 선호하는 AI 플랫폼(Claude, ChatGPT, Gemini)으로 바로 전송합니다.
  </Card>
</CardGroup>

## Append Mode

Append Mode는 페이지의 서로 다른 부분, 혹은 서로 다른 페이지에서 조각을 모아야 할 때 유용합니다.

1. 원하는 섹션을 선택하고 **Append to buffer**를 클릭합니다
2. 다른 섹션으로 스크롤해 다시 추가합니다
3. 필요한 만큼 반복합니다
4. Web2MD 팝업을 열어 합쳐진 Markdown 문서를 확인합니다

추가된 각 조각은 최종 결과물에서 수평선(`---`)으로 구분됩니다.

<Tip>
Append Mode는 긴 문서 페이지나 연구 논문에서 필요한 부분만 골라낼 때 특히 유용합니다.
</Tip>

## 스마트 선택 확장

Web2MD는 선택 영역을 완전한 구조 단위로 자동 확장합니다.

| 선택한 영역... | Web2MD가 확장하는 범위... |
|---------------|---------------------|
| 문단의 일부 | 전체 문단 |
| 코드 블록의 일부 | 언어 태그를 포함한 코드 블록 전체 |
| 목록 항목의 일부 | 완전한 목록 항목 |
| 여러 요소에 걸친 텍스트 | 완전히 또는 부분적으로 선택된 모든 블록 |

이를 통해 결과물의 Markdown이 항상 온전한 형태를 유지합니다 — 문단이 끊기거나 코드 블록이 절반만 렌더링되는 일이 없습니다.

## 호환성

선택 툴바는 모든 웹사이트에서 작동합니다. 가볍게 떠 있는 요소로 삽입되어 페이지 자체의 UI를 방해하지 않습니다.

<Info>
툴바는 텍스트를 선택했을 때만 나타납니다. 다른 곳을 클릭하거나 선택을 해제하면 자동으로 사라집니다.
</Info>
