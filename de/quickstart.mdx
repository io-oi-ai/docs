## Schritt 1: Erweiterung installieren

<Card title="Chrome Web Store" icon="chrome" href="https://chromewebstore.google.com/detail/web2md-web-to-markdown/ijmgpkkfgpijifldbjafjiapehppcbcn">
  Installieren Sie Web2MD aus dem Chrome Web Store. Es ist kostenlos – kein Konto zum Start erforderlich.
</Card>

Nach der Installation sehen Sie das Web2MD-Symbol ( **.MD** ) in Ihrer Browser-Symbolleiste. Heften Sie es an, um schnellen Zugriff zu haben.

## Schritt 2: Ihre erste Seite umwandeln

Navigieren Sie zu einer beliebigen Webseite und wählen Sie eine der folgenden Methoden:

<AccordionGroup>
  <Accordion title="Auf das Erweiterungssymbol klicken" icon="puzzle-piece">
    1. Klicken Sie auf das Web2MD-Symbol in der Symbolleiste
    2. Klicken Sie auf **Convert**
    3. Der Seiteninhalt wird sofort in Markdown umgewandelt
  </Accordion>

  <Accordion title="Tastenkombination verwenden" icon="keyboard">
    Drücken Sie `Alt+M` (oder `Option+M` auf dem Mac), um die aktuelle Seite umzuwandeln. Dies funktioniert auf jedem Tab, ohne das Popup zu öffnen.
  </Accordion>

  <Accordion title="Rechtsklick-Kontextmenü" icon="ellipsis-vertical">
    Klicken Sie mit der rechten Maustaste irgendwo auf die Seite und wählen Sie **Copy page as Markdown**. Wenn Sie Text markiert haben, wird nur die Auswahl umgewandelt.
  </Accordion>
</AccordionGroup>

## Schritt 3: Die Ausgabe nutzen

Nach der Umwandlung sehen Sie das Ergebnispanel mit mehreren Optionen:

| Aktion | Beschreibung |
|--------|-------------|
| **Copy** | Markdown in die Zwischenablage kopieren |
| **Download** | Als `.md`-Datei speichern |
| **Send to AI** | Claude, ChatGPT oder Gemini mit dem Inhalt öffnen |
| **Save to Obsidian** | An Ihren Obsidian-Vault senden (falls aktiviert) |

Außerdem sehen Sie hilfreiche Statistiken: **Wortanzahl**, **Token-Anzahl** und **geschätzte Lesezeit**.

<Tip>
Aktivieren Sie **Auto copy** in den Einstellungen, um das Markdown nach jeder Umwandlung automatisch in Ihre Zwischenablage zu kopieren.
</Tip>
