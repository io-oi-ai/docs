## Übersicht

Der Web2MD MCP-Server stellt Web2MD als eine Reihe von Tools bereit, die AI-Assistenten direkt aufrufen können, unter Verwendung des [Model Context Protocol (MCP)](https://modelcontextprotocol.io) — einem offenen Standard zur Verbindung von AI-Modellen mit externen Tools und Datenquellen.

Nach der Konfiguration können Sie Claude, Cursor oder Windsurf bitten, Seiten zu konvertieren, Ihren Konvertierungsverlauf zu durchsuchen oder gespeicherte Markdown-Inhalte abzurufen, ohne die Konversation zu verlassen.

<Note>
  Der MCP-Server erfordert einen **Web2MD API-Schlüssel** (PRO-Plan). Erstellen Sie einen im [Dashboard](https://web2md.org/dashboard/api-keys).
</Note>

## Verfügbare Tools

Der MCP-Server stellt drei Tools bereit:

| Tool | Beschreibung |
|------|-------------|
| `convert_url` | Konvertiert jede URL in sauberes Markdown |
| `semantic_search` | Durchsucht Ihre gespeicherten Konvertierungen mit natürlicher Sprache |
| `get_conversion` | Ruft das vollständige Markdown einer gespeicherten Konvertierung anhand der ID ab |

### convert_url

Ruft eine URL ab und liefert den Seiteninhalt als Markdown zurück. Unterstützt dieselben Konvertierungsoptionen wie die REST API.

**Beispiel-Prompt:** _"Convert this URL to markdown: https://docs.example.com/getting-started"_

### semantic_search

Durchsucht alle Ihre zuvor gespeicherten Konvertierungen mithilfe einer Anfrage in natürlicher Sprache. Liefert eine Liste passender Konvertierungen mit Titeln, URLs und Relevanz-Scores.

**Beispiel-Prompt:** _"Search my saved pages for articles about database indexing"_

### get_conversion

Ruft den vollständigen Markdown-Inhalt einer bestimmten gespeicherten Konvertierung anhand ihrer ID ab. Nützlich, um ein vollständiges Dokument in den Kontext zu holen, nachdem es mit `semantic_search` gefunden wurde.

**Beispiel-Prompt:** _"Get the full content of conversion abc123"_

## Einrichtung

<Tabs>
  <Tab title="Claude Desktop">
    Öffnen Sie Ihre Claude Desktop-Konfigurationsdatei:

    - **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`
    - **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

    Fügen Sie den `web2md`-Eintrag unter `mcpServers` hinzu:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Starten Sie Claude Desktop neu. Die Web2MD-Tools sollten nun im Tool-Picker erscheinen.
  </Tab>
  <Tab title="Cursor">
    Öffnen Sie Ihre Cursor MCP-Konfigurationsdatei unter `~/.cursor/mcp.json` und fügen Sie Folgendes hinzu:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Starten Sie Cursor neu, um den neuen Server zu laden.
  </Tab>
  <Tab title="Windsurf">
    Öffnen Sie Ihre Windsurf MCP-Konfigurationsdatei unter `~/.windsurf/mcp.json` und fügen Sie Folgendes hinzu:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Starten Sie Windsurf neu, um den neuen Server zu laden.
  </Tab>
</Tabs>

## Anwendungsbeispiele

Sobald der MCP-Server läuft, können Sie natürliche Sprache in Ihrem AI-Assistenten verwenden:

<AccordionGroup>
  <Accordion title="Eine Seite in Markdown konvertieren">
    **Sie:** Convert this URL to markdown: https://react.dev/learn/thinking-in-react

    **Assistent:** _(ruft `convert_url` auf)_ Hier ist das konvertierte Markdown für "Thinking in React"...
  </Accordion>
  <Accordion title="Gespeicherte Konvertierungen durchsuchen">
    **Sie:** Search my saved pages for anything about authentication

    **Assistent:** _(ruft `semantic_search` auf)_ Ich habe 3 gespeicherte Konvertierungen zum Thema Authentifizierung gefunden:
    1. "OAuth 2.0 Guide" — vor 2 Tagen gespeichert
    2. "JWT Best Practices" — letzte Woche gespeichert
    3. ...
  </Accordion>
  <Accordion title="Eine vollständige Konvertierung abrufen">
    **Sie:** Get the full content of conversion abc123

    **Assistent:** _(ruft `get_conversion` auf)_ Hier ist der vollständige Markdown-Inhalt...
  </Accordion>
</AccordionGroup>

<Tip>
  Kombinieren Sie Tools in einer einzigen Konversation. Zum Beispiel: _"Search my conversions for React docs, then get the full content of the most recent one."_
</Tip>

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Tools erscheinen nach der Einrichtung nicht">
    Stellen Sie sicher, dass Sie die Anwendung nach dem Bearbeiten der Konfigurationsdatei neu gestartet haben. Überprüfen Sie, ob `npx web2md-mcp` in Ihrem Terminal erfolgreich ausgeführt wird.
  </Accordion>
  <Accordion title="Authentifizierungsfehler">
    Überprüfen Sie, ob Ihr `WEB2MD_API_KEY` korrekt ist und Ihr PRO-Abonnement aktiv ist. Sie können Ihren Schlüssel auf der [API-Schlüssel-Seite](https://web2md.org/dashboard/api-keys) überprüfen.
  </Accordion>
  <Accordion title="npx kann das Paket nicht auflösen">
    Versuchen Sie zunächst, das Paket global zu installieren: `npm install -g web2md-mcp`, ändern Sie dann den `command` in `web2md-mcp` und entfernen Sie das `args`-Feld.
  </Accordion>
</AccordionGroup>
