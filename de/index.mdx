## Was ist Web2MD?

Web2MD ist eine Chrome-Erweiterung, die jede Webseite in sauberes Markdown-Format umwandelt. Egal ob Sie Inhalte an KI-Assistenten weitergeben, Artikel in Obsidian speichern oder Webinhalte archivieren möchten – Web2MD übernimmt die Umwandlung, damit Sie es nicht selbst tun müssen.

### Weitere Funktionen

- **KI-gestützte Zusammenfassungen und Tags** — Generiert automatisch Zusammenfassungen und Themen-Tags für jede Umwandlung (PRO)
- **Sammlungen** — Organisieren Sie gespeicherte Inhalte in Sammlungen für Ihr Wissensmanagement (PRO)
- **Semantische Suche** — Finden Sie gespeicherte Inhalte anhand ihrer Bedeutung, nicht nur nach Schlagwörtern (PRO)
- **REST API & MCP Server** — Integrieren Sie Web2MD programmatisch in Ihre Entwickler-Workflows (PRO)
- **CLI-Tool** — `npx web2md` wandelt URLs über die Kommandozeile um. Per Pipe an LLMs weiterleiten, im Batch verarbeiten oder in Obsidian-Vaults speichern
- **Agent Bridge** — Lassen Sie KI-Agenten (Claude Code, Cursor) über Ihre Chrome-Erweiterung Reddit und andere Seiten im Batch umwandeln
- **Als Bild exportieren** — Speichern Sie Markdown als gestyltes Bild mit Dark-/Light-Themes (PRO)
