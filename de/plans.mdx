## Free-Plan

Der Free-Plan umfasst **3 Umwandlungen pro Tag** — genug, um Web2MD auszuprobieren und zu sehen, ob es zu Ihrem Workflow passt.

- Umwandlungen werden serverseitig über die Web2MD-API verarbeitet
- Alle Kernfunktionen sind verfügbar (An KI senden, Batch-Umwandlung, Prompt-Vorlagen usw.)
- Keine Kreditkarte erforderlich

<Info>
Wenn Sie das kostenlose Limit erreichen, sehen Sie folgende Meldung: **"Free limit reached, please upgrade to Pro."**
</Info>

## Pro-Plan

Pro schaltet **unbegrenzte Umwandlungen** mit **client-seitiger Verarbeitung** frei – Ihre Inhalte verlassen niemals Ihren Browser.

Weitere Pro-Funktionen:

- **KI Auto-Tags & Zusammenfassung** — Jede Umwandlung generiert automatisch semantische Tags und eine prägnante Zusammenfassung
- **Markdown-Dateien herunterladen** — Speichern Sie Umwandlungen als `.md`-Dateien auf Ihrem Computer
- **Automatische Plattform-Sammlungen** — Umwandlungen werden automatisch nach Quellplattform sortiert
- **Token-Zählung** — Sehen Sie exakte Token-Anzahlen für die KI-Kontextplanung
- **Intelligente Inhaltsaufteilung** — Teilt lange Inhalte automatisch in KI-freundliche Abschnitte auf
- **KI-Zusammenfassung & Tags** — Generiert automatisch Zusammenfassungen und Themen-Tags für umgewandelte Inhalte
- **Sammlungen & semantische Suche** — Organisieren Sie gespeicherte Inhalte in Sammlungen und suchen Sie nach Bedeutung
- **Benutzerdefinierte Extraktionsregeln** — Erstellen Sie domänenspezifische CSS-Selektor-Regeln für präzise Inhaltsextraktion
- **REST-API-Zugriff** — Nutzen Sie die Konvertierungs-API von Web2MD programmatisch (60 Anfragen/Min.)
- **MCP-Server-Integration** — Verbinden Sie Web2MD mit KI-Agenten und Automatisierungstools
- **Webhook-Automatisierung** — Übertragen Sie Umwandlungsereignisse an externe Dienste
- **CLI-Tool** — Wandeln Sie URLs über die Kommandozeile mit `npx web2md` um, inklusive Batch-Modus und Obsidian-Vault-Export
- **Agent Bridge** — Lassen Sie KI-Agenten URLs über Ihre Chrome-Erweiterung per Native Messaging im Batch umwandeln
- **Als Bild exportieren** — Speichern Sie umgewandeltes Markdown als gestylte Bilder ohne Wasserzeichen
- **Watch Mode** — Überwachen Sie Seiten auf Änderungen mit automatisierter Umwandlung oder Webhook-Triggern
- **Compile Wiki** — Erstellen Sie KI-gestützte, Obsidian-fertige Wissensdatenbanken aus Sammlungen
- **API-Schlüssel-Verwaltung** — Erstellen und verwalten Sie bis zu 5 API-Schlüssel

## So funktioniert die Umwandlung je nach Plan

<AccordionGroup>
  <Accordion title="Free-Nutzer" icon="user">
    1. Site-Adapter wird ausgeführt (falls verfügbar) — dies ist kostenlos und unbegrenzt
    2. Falls kein Adapter vorhanden ist, wird das Seiten-HTML an die Web2MD-Server-API gesendet
    3. Der Server wandelt HTML in Markdown um und liefert das Ergebnis zurück
    4. Der Nutzungszähler wird erhöht
  </Accordion>

  <Accordion title="Pro-Nutzer" icon="crown">
    1. Site-Adapter wird ausgeführt (falls verfügbar)
    2. Falls kein Adapter vorhanden ist, erfolgt die Umwandlung **vollständig im Browser** (client-seitig)
    3. Es werden keine Daten an einen Server gesendet
    4. Keine Nutzungslimits
  </Accordion>
</AccordionGroup>
