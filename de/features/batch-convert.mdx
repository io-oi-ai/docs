## Überblick

Mit Batch Convert konvertieren Sie mehrere Seiten in einem einzigen Vorgang. Perfekt für Recherche-Sessions, gespeicherte Artikel oder das Archivieren mehrerer Seiten auf einmal.

## Zwei Eingabemodi

### URLs einfügen

Geben Sie mehrere URLs ein (eine pro Zeile) und konvertieren Sie sie alle auf einmal.

<Steps>
  <Step title="Den Batch-Tab öffnen">
    Klicken Sie auf den **Batch**-Tab im Web2MD-Popup.
  </Step>
  <Step title="Ihre URLs einfügen">
    Geben Sie URLs zeilenweise im Textfeld ein. Sie können aus einer Tabelle, Notizen oder einer beliebigen anderen Quelle einfügen.
  </Step>
  <Step title="Konvertierung starten">
    Klicken Sie auf **Convert All**, um zu beginnen. Jede URL wird nacheinander verarbeitet.
  </Step>
</Steps>

### Geöffnete Tabs

Wählen Sie aus den aktuell geöffneten Browser-Tabs aus, um mehrere Seiten gleichzeitig zu konvertieren.

<Steps>
  <Step title="Zum Modus Open Tabs wechseln">
    Wechseln Sie im Batch-Tab zur Ansicht **Open Tabs**.
  </Step>
  <Step title="Tabs auswählen">
    Aktivieren Sie die Tabs, die Sie konvertieren möchten. Alle konvertierbaren Tabs im aktuellen Fenster werden aufgelistet.
  </Step>
  <Step title="Konvertierung starten">
    Klicken Sie auf **Convert All**, um alle ausgewählten Tabs zu verarbeiten.
  </Step>
</Steps>

## Fortschrittsanzeige

Während der Stapelkonvertierung sehen Sie den Fortschritt in Echtzeit:

- **Fortschrittsbalken** mit prozentualer Fertigstellung
- Individueller Status für jede URL:

| Status | Bedeutung |
|--------|---------|
| Pending | Wartet auf Verarbeitung |
| Converting | Wird gerade konvertiert |
| Success | Konvertierung abgeschlossen |
| Error | Konvertierung fehlgeschlagen (z. B. Seite nicht erreichbar) |
| Skipped | URL war ungültig oder nicht unterstützt |

## Ergebnisse

Nach Abschluss sehen Sie eine Zusammenfassung:

- Gesamtzahl der **erfolgreichen** Konvertierungen
- Gesamtzahl der **fehlgeschlagenen** / **übersprungenen** Konvertierungen
- Aktionen: **Copy All** (alle Ergebnisse kombiniert) oder **Download All** (einzelne `.md`-Dateien)

Klicken Sie auf **New Batch**, um eine weitere Stapelkonvertierung zu starten.

<Tip>
Die Stapelkonvertierung funktioniert am besten mit Artikelseiten und Blogbeiträgen. Dynamische Single-Page-Anwendungen können weniger konsistente Ergebnisse liefern.
</Tip>
