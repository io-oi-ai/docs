## Überblick

Wenn Sie eine Seite konvertieren, generiert Web2MD automatisch:
- **1-2 Sätze Zusammenfassung** — Ein prägnanter Überblick über den Inhalt
- **3-5 semantische Tags** — Kleingeschriebene englische Tags zur Kategorisierung

Dies geschieht im Hintergrund (~1-2 Sekunden), angetrieben vom Doubao-KI-Modell.

## So funktioniert es

<Steps>
  <Step title="Konvertieren Sie eine Seite">
    Konvertieren Sie eine beliebige Webseite mit Web2MD.
  </Step>
  <Step title="KI analysiert den Inhalt">
    Die ersten ~4000 Zeichen werden zur Analyse an das KI-Modell gesendet.
  </Step>
  <Step title="Tags erscheinen">
    Zusammenfassung und Tags werden unterhalb des Konvertierungsergebnisses als farbige Pillen angezeigt.
  </Step>
</Steps>

## Filtern nach Tags

Im **History**-Tab können Sie Ihre Konvertierungen filtern, indem Sie auf einen Tag klicken. So finden Sie schnell alle Artikel zu einem bestimmten Thema.

## PRO-Funktion

<Info>
KI-Tags und Zusammenfassung sind eine **PRO**-Funktion. Kostenlose Nutzer sehen eine unscharfe Vorschau mit einem Upgrade-Hinweis.
</Info>
