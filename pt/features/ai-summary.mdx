## Visão geral

A partir da v0.6, cada conversão gera automaticamente um resumo curto e um conjunto de tags semânticas. Sem necessidade de configuração — o Web2MD cuida de tudo nos bastidores, para que você consiga navegar pelo seu histórico rapidamente.

<Note>
AI Summary & Tags é um recurso **exclusivo do plano Pro**. Usuários do plano gratuito veem uma prévia desfocada com um aviso de upgrade.
</Note>

## Como funciona

<Steps>
  <Step title="Converta uma página">
    Converta qualquer página da web usando o popup, o atalho de teclado ou o menu de contexto — o fluxo de sempre.
  </Step>
  <Step title="O resumo é gerado automaticamente">
    Após a conversão, o Claude Haiku gera um resumo de 1 a 2 frases e de 3 a 5 tags semânticas com base no conteúdo extraído.
  </Step>
  <Step title="Veja no Preview Card ou no Histórico">
    O resumo e as tags aparecem diretamente no Preview Card e na sua lista de Histórico.
  </Step>
</Steps>

## O que você recebe

<CardGroup cols={2}>
  <Card title="Smart Summary" icon="align-left">
    Um resumo conciso de 1 a 2 frases do conteúdo da página. Útil para relembrar rapidamente do que tratava um artigo, sem precisar reler tudo.
  </Card>
  <Card title="Semantic Tags" icon="tags">
    De 3 a 5 tags geradas automaticamente que capturam os temas principais. Clique em qualquer tag para filtrar seu Histórico por aquele tópico.
  </Card>
</CardGroup>

## Onde os resumos aparecem

| Local | O que você vê |
|----------|-------------|
| **Preview Card** | Resumo abaixo da barra de estatísticas, tags como pílulas clicáveis |
| **History list** | Trecho do resumo abaixo de cada item, tags para filtragem rápida |
| **History search** | Clique em uma tag para encontrar todas as conversões daquele tópico |

## Free vs Pro

<Tabs>
  <Tab title="Usuários Pro">
    O resumo e as tags completos são exibidos em cada conversão. Clique em qualquer tag para ir a uma visualização filtrada do Histórico com todas as conversões correspondentes.
  </Tab>
  <Tab title="Usuários Free">
    O resumo e as tags são gerados, mas exibidos como uma **prévia desfocada**. Clique na área desfocada para ver um aviso de upgrade.
  </Tab>
</Tabs>

## Zero configuração

Não há nada para configurar. Você não precisa fornecer uma chave de API nem ativar nenhuma opção — o AI Summary & Tags roda automaticamente em cada conversão para usuários Pro.

<Tip>
As tags são uma forma rápida de construir uma base de conhecimento pesquisável. Com o tempo, seu Histórico se torna uma biblioteca organizada por tópicos de tudo o que você já converteu.
</Tip>
