## O que é o Web2MD?

O Web2MD é uma extensão do Chrome que converte qualquer página da web em Markdown limpo. Seja para alimentar assistentes de IA com conteúdo, salvar artigos no Obsidian ou arquivar conteúdo da web — o Web2MD cuida da conversão para você não precisar.

### Mais funcionalidades

- **Resumos e tags com IA** — Gere automaticamente resumos e tags de tópicos para cada conversão (PRO)
- **Coleções** — Organize o conteúdo salvo em coleções para gestão de conhecimento (PRO)
- **Busca semântica** — Encontre conteúdo salvo pelo significado, não apenas por palavras-chave (PRO)
- **API REST e servidor MCP** — Integre o Web2MD aos seus fluxos de trabalho de desenvolvimento de forma programática (PRO)
- **Ferramenta CLI** — `npx web2md` converte URLs pela linha de comando. Envie para LLMs, processe em lote ou salve em vaults do Obsidian
- **Agent Bridge** — Deixe agentes de IA (Claude Code, Cursor) converterem em lote conteúdo do Reddit e outros sites pela sua extensão do Chrome
- **Exportar como imagem** — Salve o Markdown como uma imagem estilizada com temas claro/escuro (PRO)
