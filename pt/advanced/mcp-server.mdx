## Visão geral

O servidor MCP do Web2MD expõe o Web2MD como um conjunto de ferramentas que assistentes de IA podem chamar diretamente, usando o [Model Context Protocol (MCP)](https://modelcontextprotocol.io) — um padrão aberto para conectar modelos de IA a ferramentas e fontes de dados externas.

Depois de configurado, você pode pedir ao Claude, Cursor ou Windsurf para converter páginas, buscar no seu histórico de conversões ou recuperar Markdown salvo sem sair da conversa.

<Note>
  O servidor MCP requer uma **chave de API do Web2MD** (plano PRO). Gere uma no [painel](https://web2md.org/dashboard/api-keys).
</Note>

## Ferramentas disponíveis

O servidor MCP oferece três ferramentas:

| Ferramenta | Descrição |
|------|-------------|
| `convert_url` | Converte qualquer URL em Markdown limpo |
| `semantic_search` | Busca nas suas conversões salvas usando linguagem natural |
| `get_conversion` | Recupera o Markdown completo de uma conversão salva pelo ID |

### convert_url

Busca uma URL e retorna o conteúdo da página como Markdown. Suporta as mesmas opções de conversão da REST API.

**Exemplo de prompt:** _"Convert this URL to markdown: https://docs.example.com/getting-started"_

### semantic_search

Busca em todas as suas conversões salvas anteriormente usando uma consulta em linguagem natural. Retorna uma lista de conversões correspondentes com títulos, URLs e pontuações de relevância.

**Exemplo de prompt:** _"Search my saved pages for articles about database indexing"_

### get_conversion

Recupera o conteúdo completo em Markdown de uma conversão salva específica pelo seu ID. Útil para trazer um documento completo para o contexto depois de encontrá-lo com `semantic_search`.

**Exemplo de prompt:** _"Get the full content of conversion abc123"_

## Configuração

<Tabs>
  <Tab title="Claude Desktop">
    Abra o arquivo de configuração do Claude Desktop:

    - **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`
    - **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

    Adicione a entrada `web2md` em `mcpServers`:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Reinicie o Claude Desktop. Você deve ver as ferramentas do Web2MD aparecerem no seletor de ferramentas.
  </Tab>
  <Tab title="Cursor">
    Abra o arquivo de configuração MCP do Cursor em `~/.cursor/mcp.json` e adicione:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Reinicie o Cursor para carregar o novo servidor.
  </Tab>
  <Tab title="Windsurf">
    Abra o arquivo de configuração MCP do Windsurf em `~/.windsurf/mcp.json` e adicione:

    ```json
    {
      "mcpServers": {
        "web2md": {
          "command": "npx",
          "args": ["web2md-mcp"],
          "env": {
            "WEB2MD_API_KEY": "w2m_your_key_here"
          }
        }
      }
    }
    ```

    Reinicie o Windsurf para carregar o novo servidor.
  </Tab>
</Tabs>

## Exemplos de uso

Com o servidor MCP em execução, você pode usar linguagem natural no seu assistente de IA:

<AccordionGroup>
  <Accordion title="Converter uma página em Markdown">
    **Você:** Convert this URL to markdown: https://react.dev/learn/thinking-in-react

    **Assistente:** _(chama `convert_url`)_ Aqui está o Markdown convertido de "Thinking in React"...
  </Accordion>
  <Accordion title="Buscar conversões salvas">
    **Você:** Search my saved pages for anything about authentication

    **Assistente:** _(chama `semantic_search`)_ Encontrei 3 conversões salvas relacionadas a autenticação:
    1. "OAuth 2.0 Guide" — salva há 2 dias
    2. "JWT Best Practices" — salva na semana passada
    3. ...
  </Accordion>
  <Accordion title="Recuperar uma conversão completa">
    **Você:** Get the full content of conversion abc123

    **Assistente:** _(chama `get_conversion`)_ Aqui está o conteúdo completo em Markdown...
  </Accordion>
</AccordionGroup>

<Tip>
  Combine ferramentas em uma única conversa. Por exemplo: _"Search my conversions for React docs, then get the full content of the most recent one."_
</Tip>

## Solução de problemas

<AccordionGroup>
  <Accordion title="As ferramentas não aparecem depois da configuração">
    Certifique-se de ter reiniciado o aplicativo depois de editar o arquivo de configuração. Verifique se `npx web2md-mcp` executa com sucesso no seu terminal.
  </Accordion>
  <Accordion title="Erros de autenticação">
    Confira se sua `WEB2MD_API_KEY` está correta e se sua assinatura PRO está ativa. Você pode verificar sua chave na [página de chaves de API](https://web2md.org/dashboard/api-keys).
  </Accordion>
  <Accordion title="npx não consegue resolver o pacote">
    Tente instalar o pacote globalmente primeiro: `npm install -g web2md-mcp`, depois altere o `command` para `web2md-mcp` e remova o campo `args`.
  </Accordion>
</AccordionGroup>
