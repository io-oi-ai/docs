## Passo 1: Instale a extensão

<Card title="Chrome Web Store" icon="chrome" href="https://chromewebstore.google.com/detail/web2md-web-to-markdown/ijmgpkkfgpijifldbjafjiapehppcbcn">
  Instale o Web2MD na Chrome Web Store. É gratuito — não é preciso criar conta para começar.
</Card>

Depois da instalação, você verá o ícone do Web2MD ( **.MD** ) na barra de ferramentas do navegador. Fixe-o para facilitar o acesso.

## Passo 2: Converta sua primeira página

Navegue até qualquer página da web e escolha um destes métodos:

<AccordionGroup>
  <Accordion title="Clique no ícone da extensão" icon="puzzle-piece">
    1. Clique no ícone do Web2MD na barra de ferramentas
    2. Clique em **Convert**
    3. O conteúdo da página é convertido para Markdown instantaneamente
  </Accordion>

  <Accordion title="Use o atalho de teclado" icon="keyboard">
    Pressione `Alt+M` (ou `Option+M` no Mac) para converter a página atual. Funciona em qualquer aba, sem precisar abrir o popup.
  </Accordion>

  <Accordion title="Menu de contexto (botão direito)" icon="ellipsis-vertical">
    Clique com o botão direito em qualquer lugar da página e selecione **Copy page as Markdown**. Se você tiver um texto selecionado, apenas a seleção será convertida.
  </Accordion>
</AccordionGroup>

## Passo 3: Use o resultado

Depois da conversão, você verá o painel de resultado com várias opções:

| Ação | Descrição |
|--------|-------------|
| **Copy** | Copia o Markdown para a área de transferência |
| **Download** | Salva como um arquivo `.md` |
| **Send to AI** | Abre o Claude, ChatGPT ou Gemini com o conteúdo |
| **Save to Obsidian** | Envia para o seu vault do Obsidian (se ativado) |

Você também verá estatísticas úteis: **contagem de palavras**, **contagem de tokens** e **tempo estimado de leitura**.

<Tip>
Ative **Auto copy** nas configurações para copiar o Markdown automaticamente para a área de transferência após cada conversão.
</Tip>
